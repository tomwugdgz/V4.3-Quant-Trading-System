## 💻 使用方法

### 方式 1: 手动运行

```bash
cd C:\Users\DELL\.openclaw-autoclaw\workspace\trading
python wangcai_stock_screener.py
```

### 方式 2: 定时任务 (Windows)

1. 打开"任务计划程序"
2. 创建基本任务 → "旺财选股"
3. 触发器：每周一 09:00
4. 操作：启动程序
   - 程序：`python.exe`
   - 参数：`wangcai_stock_screener.py`
   - 起始于：`C:\Users\DELL\.openclaw-autoclaw\workspace\trading`

### 方式 3: Python 定时脚本

```python
import schedule
import time

def job():
    import subprocess
    subprocess.run(['python', 'wangcai_stock_screener.py'])

# 每周一 09:00 运行
schedule.every().monday.at("09:00").do(job)

while True:
    schedule.run_pending()
    time.sleep(60)
```

## ⚙️ 配置参数

编辑 `wangcai_stock_screener.py` 中的 `Config` 类：

```python
class Config:
    # 1. 毛利率
    MIN_GROSS_MARGIN = 20.0  # 最低 20%
    
    # 2. 股本 (亿股)
    MAX_TOTAL_SHARE = 5.0  # 最大 5 亿股
    
    # 3. ROE
    MIN_ROE = 5.0  # 最低 5%
    
    # 4. 主营业务占比
    MIN_MAIN_BUSINESS_RATIO = 70.0  # 最低 70%
    
    # 5. 股价位置
    MAX_PRICE_MA_RATIO = 1.1  # 股价/均线 < 1.1
    
    # 6. 行业壁垒
    MIN_BARRIER_SCORE = 3  # 最低 3 星
    
    # 7. 上涨空间
    MIN_UPSIDE = 20.0  # 最低 20%
    
    # 8. 稀缺性
    MIN_RARITY_SCORE = 4  # 最低 4 星
    
    # 9. 股东架构
    PREFERRED_SHAREHOLDER_TYPES = ['创始人', '产业资本', '国资']
    
    # 10. 派息率
    MIN_DIVIDEND_YIELD = 2.0  # 最低股息率 2%
```

## 📈 评分体系

### 总分 = 100 分

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| 毛利率 | 10 分 | >40%=10 分，30-40%=8 分，20-30%=6 分 |
| 股本 | 10 分 | <1 亿=10 分，1-3 亿=8 分，3-5 亿=6 分 |
| ROE | 15 分 | >20%=15 分，15-20%=12 分，10-15%=8 分，5-10%=4 分 |
| 主营业务占比 | 10 分 | >90%=10 分，70-90%=8 分 |
| 股价位置 | 15 分 | MA 附近=15 分，略高=8 分 |
| 行业壁垒 | 10 分 | 5 星=10 分，4 星=8 分，3 星=6 分 |
| 上涨空间 | 10 分 | >50%=10 分，30-50%=8 分，20-30%=6 分 |
| 稀缺性 | 10 分 | 5 星=10 分，4 星=8 分 |
| 股东架构 | 10 分 | 创始人/产业=10 分，国资=8 分 |
| 派息率 | 10 分 | >5%=10 分，3-5%=8 分，2-3%=6 分 |

### 评级

| 分数 | 评级 | 操作建议 |
|------|------|----------|
| 90-100 | ⭐⭐⭐⭐⭐ 强烈推荐 | 重点配置 |
| 80-89 | ⭐⭐⭐⭐ 推荐 | 适当配置 |
| 70-79 | ⭐⭐⭐ 关注 | 观察等待 |
| 60-69 | ⭐⭐ 观察 | 暂不介入 |
| <60 | ❌ 淘汰 | 不考虑 |
