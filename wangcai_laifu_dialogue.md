### 问题 2: "半仓"定义

**半仓 = 标准仓位的 50%**

```
标准仓位 = 账户风险 0.5% / 止损幅度

示例:
- 账户 $10,000，单笔风险 0.5% = $50
- 止损 30 pips，每点$1 → 止损幅度$30
- 标准仓位 = $50 / $30 = 1.67 手 → 实际 1.5 手

Tier 1 (≥75 分): 1.5 手 (100%)
Tier 2 (60-74 分): 0.75 手 (50%)
Tier 3 (45-59 分): 观察，不直接开仓
```

**关键**: Tier 2 需要二次确认（价格回调或突破关键位）

```python
def is_ranging_market(symbol, timeframe):
    adx = get_adx(symbol, timeframe, period=14)
    bb_width = get_bollinger_band_width(symbol, timeframe)
    bb_width_avg = average(bb_width, 20)
    
    # 主条件
    if adx < 20 and bb_width < bb_width_avg * 0.8:
        return True
    
    # 辅助条件 (ADX 略高但带宽极窄)
    if adx < 25 and bb_width < bb_width_avg * 0.6:
        return True
    
    return False
```

### 回应 2: 动量因子反向公式

**你的公式对，但需要平滑处理**:

```python
# 震荡市动量因子修正
def adjust_momentum_for_ranging(momentum_score, adx):
    # ADX 越低，反向力度越大
    reverse_factor = (25 - adx) / 25  # ADX=20 时 0.2, ADX=10 时 0.6
    
    # 平滑反向
    if momentum_score > 50:  # 原为正向强势
        adjusted = 50 - (momentum_score - 50) * reverse_factor
    else:  # 原为弱势
        adjusted = 50 + (50 - momentum_score) * reverse_factor
    
    return max(0, min(100, adjusted))
```

**示例**:
- 原始动量 80 分，ADX=15 → 修正后 32 分 (反向)
- 原始动量 30 分，ADX=15 → 修正后 68 分 (超卖反弹)
