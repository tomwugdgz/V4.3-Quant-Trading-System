## 二、模块测试（Module Testing）

### 2.1 Market Regime 测试

- [ ] 能正确判断市场状态
- [ ] 动态阈值生成正常
- [ ] 仓位系数计算正确

**测试命令**:
```bash
cd v4_3
python market_regime.py
```

**验收标准**:
- 输出包含 6+ 个品种的市场状态
- 每个品种有明确的状态（TRENDING_UP/RANGING 等）
- 无报错

### 2.2 Factor Score 测试

- [ ] 4 因子计算正常
- [ ] 综合评分在 0-100 范围
- [ ] 信号映射正确

**测试命令**:
```bash
python factor_score.py
```

**验收标准**:
- 每个品种有 4 个因子值
- 综合评分合理（50 左右为中性）
- 信号类型正确（BUY/SELL/NEUTRAL）

### 2.3 Risk Agent 测试

- [ ] 5 层风控检查正常
- [ ] 仓位计算正确
- [ ] VaR 计算正常

**测试命令**:
```bash
python risk_agent.py
```

**验收标准**:
- 风控报告包含所有检查项
- 保证金水平、杠杆显示正确
- 无报错

### 2.4 Walk-Forward 测试

- [ ] 滚动窗口生成正确
- [ ] 参数优化正常
- [ ] IC 计算正常

**测试命令**:
```bash
python walk_forward.py
python factor_ic.py
```

**验收标准**:
- 生成 3+ 个滚动窗口
- 输出综合评分
- IC 分析包含显著性检验
