## ⚠️ 发现的问题

### 1. 止损止盈计算问题
**问题**: EURUSD 的止损止盈计算错误
- 当前逻辑：`sl = price - sl_points * 0.01`
- 问题：不同品种的点值不同，不能统一用 0.01

**修复方案**:
```python
# 获取品种信息
symbol_info = mt5.symbol_info(symbol)
point = symbol_info.point  # 获取正确的点值

# 根据方向计算
if order_type == 'BUY':
    sl = price - sl_points * point
    tp = price + tp_points * point
else:
    sl = price + sl_points * point
    tp = price - tp_points * point
```

### 2. 平仓盈亏获取
**问题**: `OrderSendResult` 对象没有 `profit` 属性
**修复**: 平仓后需要查询账户余额变化来计算盈亏

### 3. 点差成本
**观察**: 每笔交易约有 $0.24 的点差成本
**建议**: 实盘交易时需要考虑点差对策略的影响
