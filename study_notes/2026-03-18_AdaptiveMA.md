## 一、Adaptive Moving Average (AMA) 核心概念

### 什么是自适应均线？

**传统均线的问题**:
- 简单均线 (SMA) 和指数均线 (EMA) 参数固定
- 趋势行情：反应太慢，滞后严重
- 震荡行情：频繁假信号，来回打脸

**AMA 的解决方案**:
- 根据市场波动率自动调整灵敏度
- 趋势明显时 → 快速响应 (短周期)
- 震荡盘整时 → 降低灵敏度 (长周期)

## 二、AMA 计算公式

### 核心参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ER 周期 | 10 | 效率比率计算周期 |
| 快线 SC | 2/11 | 快速平滑常数 (约等于 11 周期 EMA) |
| 慢线 SC | 2/31 | 慢速平滑常数 (约等于 31 周期 EMA) |

### 计算步骤

```
1. 计算效率比率 (ER)
   ER = |当前价 - N 周期前价| / Σ|每根 K 线变化|
   ER 范围：0 (完全震荡) ~ 1 (单边趋势)

2. 计算平滑常数 (SC)
   SC = ER × (快线 SC - 慢线 SC) + 慢线 SC
   SC 范围：慢线 SC ~ 快线 SC

3. 计算 AMA
   AMA = 前一根 AMA + SC × (当前价 - 前一根 AMA)
```

## 三、交易信号规则

### 多头信号

| 条件 | 说明 |
|------|------|
| 价格 > AMA | 价格在均线上方 |
| AMA 向上倾斜 | 均线方向向上 |
| ER > 0.5 | 市场处于趋势状态 |
| 前一根 K 线价格 < AMA | 刚突破均线 |

### 空头信号

| 条件 | 说明 |
|------|------|
| 价格 < AMA | 价格在均线下方 |
| AMA 向下倾斜 | 均线方向向下 |
| ER > 0.5 | 市场处于趋势状态 |
| 前一根 K 线价格 > AMA | 刚跌破均线 |

### 过滤条件 (避免假信号)

| 过滤器 | 参数 | 作用 |
|--------|------|------|
| ADX | > 25 | 确保趋势强度 |
| ATR | > 平均 ATR 的 80% | 确保波动率足够 |
| 时间过滤 | 避开重大数据发布 | 减少噪音 |
| 多品种确认 | 相关品种同向 | 增加胜率 |

## 四、止损止盈设置

### 止损方法

| 方法 | 参数 | 适用场景 |
|------|------|----------|
| ATR 止损 | 1.5-2×ATR | 通用 |
| AMA 止损 | 价格跌破/突破 AMA | 趋势跟踪 |
| 固定点数 | 根据品种设定 | 短线交易 |
| 百分比止损 | 账户 2% | 资金管理 |

### 止盈方法

| 方法 | 参数 | 说明 |
|------|------|------|
| 固定盈亏比 | 1:1.5 ~ 1:3 | 风险回报比 |
| 移动止损 | AMA 反向突破 | 让利润奔跑 |
| 分批止盈 | 50% @ 1R, 50% 追踪 | 锁定部分利润 |
| 前高/前低 | 关键支撑阻力 | 技术位止盈 |

## 五、多品种策略

### 推荐品种组合

| 类别 | 品种 | 特点 |
|------|------|------|
| 外汇 | EURUSD, GBPUSD, USDJPY | 流动性好，点差低 |
| 商品 | GOLD, SILVER, CRUDE | 趋势性强 |
| 股指 | US30, SPX500, NAS100 | 波动大 |
|  crypto | BTCUSD, ETHUSD | 高波动，高风险 |

### 相关性管理

| 正相关 | 负相关 | 不相关 |
|--------|--------|--------|
| EURUSD / GBPUSD | EURUSD / USDCHF | EURUSD / GOLD |
| AUDUSD / NZDUSD | USDJPY / 日经 225 | GOLD / USDJPY |

**原则**: 同一方向持仓不超过 2 个正相关品种

## 六、策略优势

| 优势 | 说明 |
|------|------|
| 自适应 | 根据市场状态自动调整 |
| 滞后小 | 趋势行情反应快 |
| 假信号少 | 震荡行情过滤好 |
| 多品种 | 适用于各类交易品种 |
| 参数少 | 易于理解和优化 |

## 七、策略劣势

| 劣势 | 解决方案 |
|------|----------|
| 极端震荡仍会亏损 | 加 ADX 过滤，ER < 0.3 不交易 |
| 趋势反转时滞后 | 配合价格行为确认 |
| 参数需要优化 | 不同品种测试最佳参数 |
| 不适合超短线 | 最小时间框架 H1 |

## 九、如何应用到我们的交易

### 改进现有策略

**当前问题**:
- 只用简单均线 (5/20)
- 震荡行情频繁止损
- 趋势反转反应慢

**AMA 改进方案**:

| 项目 | 当前 | 改进后 |
|------|------|--------|
| 趋势判断 | 5/20 均线交叉 | AMA 方向 + ER 确认 |
| 入场信号 | 均线交叉 | 价格突破 AMA + ER > 0.5 |
| 震荡过滤 | 无 | ER < 0.3 不交易 |
| 止损 | 1.5×ATR | AMA 反向突破 + ATR |
| 止盈 | 2.5×ATR | 移动止损 (AMA 追踪) |

## 十、下次交易执行计划

### 入场条件 (做多示例)

```
1. AMA 向上倾斜
2. 价格 > AMA
3. ER > 0.5 (趋势确认)
4. ADX > 25 (趋势强度)
5. 价格回踩 AMA 不破 (精确入场)
6. 止损：AMA 下方 1×ATR
7. 止盈：第一目标 2R，剩余追踪 AMA
```

### 仓位计算

```
风险金额 = 账户 × 2%
止损距离 = 入场价 - AMA(或 ATR 倍数)
仓位 = 风险金额 / (止损距离 × 每点价值)
```

## 十二、学习总结

**核心收获**:
1. AMA 比传统均线更智能，自适应市场状态
2. ER 效率比率是核心，区分趋势和震荡
3. 必须配合过滤器 (ADX、ATR) 使用
4. 多品种分散风险，但要注意相关性
5. 止损止盈要灵活，趋势行情让利润奔跑
