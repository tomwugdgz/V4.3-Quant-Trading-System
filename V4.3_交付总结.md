## 二、核心功能

### 2.1 Market Regime Agent

**解决问题**: "策略结构偏随机"

**功能**:
- ✅ 判断 4 种市场状态（趋势/震荡/高波动/未知）
- ✅ 动态调整信号阈值（0.08%-0.20%）
- ✅ 动态仓位系数（0.3x-1.0x）
- ✅ 市场状态报告生成

**核心算法**:
```python
市场状态 = f(ADX, EMA, ATR, 布林带)

TRENDING_UP: ADX > 25 AND EMA10 > EMA20
TRENDING_DOWN: ADX > 25 AND EMA10 < EMA20
RANGING: ADX < 20 AND 布林带收口
HIGH_VOLATILITY: ATR > 1.5 × 平均 ATR
```

---

### 2.2 Factor Score Engine

**解决问题**: "无稳定 Edge"

**功能**:
- ✅ 4 因子评分体系（0-100 分）
- ✅ 多因子加权综合
- ✅ 信号映射（STRONG_BUY/BUY/NEUTRAL/SELL/STRONG_SELL）
- ✅ 因子值历史记录

**因子库**:
1. **动量因子** (30%) - EMA 斜率 + 价格动量
2. **均值回归** (30%) - RSI + 布林带位置
3. **突破因子** (20%) - N 日突破 + 成交量
4. **波动率因子** (20%) - ATR 变化 + 布林带宽度

**评分映射**:
```python
score >= 70: STRONG_BUY
55 <= score < 70: BUY
45 < score < 55: NEUTRAL
30 < score <= 45: SELL
score <= 30: STRONG_SELL
```

---

### 2.3 Risk Agent

**解决问题**: "风控过强压制盈利" → "动态风控"

**功能**:
- ✅ 5 层风控检查
- ✅ 实时否决权
- ✅ VaR 计算（历史模拟法）
- ✅ 动态仓位计算
- ✅ 相关性检查

**风控检查**:
1. 账户级（保证金水平≥200%，杠杆≤3x）
2. 仓位级（总持仓≤3，单一品种≤1）
3. 风险敞口（总风险≤3%）
4. VaR（95% VaR ≤ 2% 净值）
5. 相关性（避免同向叠加）

**仓位计算**:
```python
单笔风险 = 0.5% 账户
止损距离 = 1.5 × ATR
手数 = 风险金额 / (止损距离 × 点值)
```

---

### 2.4 Review Agent

**解决问题**: "复盘靠人工"

**功能**:
- ✅ 交易归因分析（策略/品种/时间/因子）
- ✅ 每日复盘报告（Markdown 格式）
- ✅ 交易模式识别
- ✅ 问题诊断与改进建议

**归因维度**:
1. 策略归因 - 按订单类型/策略分组
2. 品种归因 - 各币种盈亏贡献
3. 时间归因 - 亚盘/欧盘/美盘表现
4. 因子归因 - 各因子 IC 变化（需数据支持）

### 3.2 配置调整

**市场状态参数** (`config/regime_config.json`):
```json
{
  "adx_threshold": 25,        // ADX 趋势阈值
  "atr_multiplier": 1.5,      // ATR 波动倍数
  "dynamic_thresholds": {     // 动态信号阈值
    "TRENDING_UP": 0.08,
    "RANGING": 0.15
  }
}
```

**因子权重** (`config/factor_weights.json`):
```json
{
  "factors": {
    "momentum": 0.30,
    "mean_reversion": 0.30,
    "breakout": 0.20,
    "volatility": 0.20
  }
}
```

**风控参数** (`config/risk_params.json`):
```json
{
  "max_leverage": 3.0,
  "max_total_risk": 0.03,
  "position_sizing": {
    "risk_per_trade": 0.005
  }
}
```
