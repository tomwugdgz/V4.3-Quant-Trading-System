## 二、优化方案

### 2.1 修复信号强度计算 ✅

**文件**: `trading/find_opportunity.py`

**修复前**:
```python
signal_strength = base_strength * (confirmations / 4)
# 问题：可能超过 1%，实际出现 30% 异常值
```

**修复后**:
```python
raw_strength = base_strength * (confirmations / 4)
signal_strength = min(raw_strength, 1.0)  # 归一化到 0-1%

# 异常值检测
if raw_strength > 1.0:
    print(f"⚠️ WARNING: {symbol} raw strength {raw_strength:.2f}% capped to 1.0%")
```

**效果**: 信号强度上限 1%，杜绝 30% 异常值

---

### 2.2 单一品种风险限制 ✅

**文件**: `trading/risk_manager_v2.py` + `trading/auto_monitor.py`

**新规则**:
```python
MAX_SINGLE_SYMBOL_RISK = 1.0  # 单一品种最大风险 1% 账户
MAX_POSITIONS_PER_SYMBOL = 1  # 单一品种最多 1 单
```

**检查逻辑**:
```python
def check_symbol_exposure(symbol, positions):
    symbol_positions = [p for p in positions if p['symbol'] == symbol]
    
    if len(symbol_positions) >= MAX_POSITIONS_PER_SYMBOL:
        return False, f"{symbol} 已有 {len(symbol_positions)} 单持仓 (上限 {MAX_POSITIONS_PER_SYMBOL})"
    
    return True, "通过"
```

**效果**: 避免 3 单 NZDUSD 的悲剧重演

---

### 2.3 三层移动止损 ✅

**文件**: `trading/risk_manager_v2.py`

**规则**:
```python
TRAILING_STOP_LAYERS = {
    'layer1': {'profit_pct': 0.3, 'action': 'breakeven'},  # 盈利 0.3% → 移至保本
    'layer2': {'profit_pct': 0.6, 'action': 'lock_50'},    # 盈利 0.6% → 锁定 50%
    'layer3': {'profit_pct': 1.0, 'action': 'lock_80'},    # 盈利 1.0% → 锁定 80%
}
```

**执行逻辑**:
- ✅ 盈利 0.3%: 止损移至保本 → 这单至少不亏
- ✅ 盈利 0.6%: 锁定 50% 利润 → 继续奔跑
- ✅ 盈利 1.0%: 锁定 80% 利润 → 让利润最大化

**效果**: 盈利单不会变亏损，亏损单严格止损

---

### 2.4 保证金自动平仓 ✅

**文件**: `trading/risk_manager_v2.py`

**规则**:
```python
MARGIN_LEVEL_CRITICAL = 150  # 危险线：强制平仓
MARGIN_LEVEL_WARNING = 200  # 警告线：禁止开新仓
```

**执行逻辑**:
```python
if margin_level < MARGIN_LEVEL_CRITICAL:
    close_all_positions(f"保证金水平过低：{margin_level:.1f}%")
    return "CRITICAL"
```

**效果**: 避免保证金从 523% 跌至 131% 的失控
