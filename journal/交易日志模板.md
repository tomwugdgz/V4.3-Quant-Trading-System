## 🔧 使用指南

### 每笔交易后
1. 立即填写基本信息
2. 记录入场理由 (信号)
3. 设置止损止盈
4. 出场后填写结果
5. 写反思和学习

### 每日复盘会
1. 汇总当日交易
2. Agent 点评
3. 总结最佳/最差
4. 提取关键学习

### 每周回顾
1. 统计周数据
2. 分析胜率/盈亏比
3. 识别模式
4. 调整策略

---

## 📊 数据分析

### 关键指标

| 指标 | 目标 | 计算方式 |
|------|------|----------|
| 胜率 | ≥55% | 盈利交易数 / 总交易数 |
| 盈亏比 | ≥1:2 | 平均盈利 / 平均亏损 |
| 期望值 | >0 | 胜率×盈亏比 - (1-胜率) |
| 最大回撤 | <20% | 最大连续亏损 |
| 夏普比率 | >1 | (收益 - 无风险) / 波动率 |

### 改进循环
```
记录 → 分析 → 学习 → 改进 → 记录
```

## 🎯 示例

```markdown
## 交易 #1

### 基本信息
- **时间**: 2026-03-19 15:30
- **货币对**: EURUSD
- **方向**: LONG
- **仓位**: 0.30 手
- **风险**: 2% ($188)

### 入场
- **价格**: 1.0950
- **时间**: 15:30
- **理由**: 
  - AMA 向上 (主信号)
  - ADX 28 (>25)
  - RSRS 0.87 (>0.8)
  - 价格突破前高

### 止损止盈
- **止损**: 1.0900 (-50 点，-$188)
- **止盈 1**: 1.1050 (+100 点，+$376)
- **止盈 2**: 1.1100 (+150 点，+$564)

### 出场
- **出场时间**: 18:45
- **出场价格**: 1.1050
- **出场原因**: 止盈 1 触发 (50% 仓位)
- **盈亏**: +$188 (50% 仓位)

### 反思
- **做得好**: 严格执行信号，耐心等确认
- **需改进**: 可以等回调再入场
- **学习**: 四重确认胜率高

### Agent 点评
**巴菲特**: "好孩子，风险控制得很好" (评分：9/10)  
**芒格**: "决策过程清晰，但为什么选 EURUSD？" (评分：8/10)  
**索罗斯**: "市场看多情绪正确，继续持有" (评分：9/10)
```
