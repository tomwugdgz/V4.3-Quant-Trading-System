## 一、系统概述

V4.3 是一个多因子量化交易系统，通过市场状态判断、多因子评分、动态风控三大核心模块，实现智能化交易决策。

### 1.1 设计目标

- **稳定性**: 从 v2.0 的 45% 回测稳定性提升至≥70%
- **Edge**: 从 v2.0 的 72.9 综合评分提升至≥80
- **胜率**: 从 v2.0 的 52.8% 提升至≥58%
- **风控**: 从"硬编码"升级为"动态风控"

### 1.2 核心特性

1. **市场状态感知**: 根据趋势/震荡/高波动动态调整策略
2. **多因子评分**: 4 大因子库，16 个子因子，全面评估市场
3. **动态风控**: 独立 Risk Agent，实时否决权
4. **自动化复盘**: 自动归因分析，持续进化

## 四、核心模块详解

### 4.1 Market Regime Agent

**文件**: `v4_3/market_regime.py`

**功能**:
- 判断市场状态（趋势/震荡/高波动）
- 动态调整信号阈值
- 决定是否可以交易

**市场状态**:
```python
class MarketRegime(Enum):
    TRENDING_UP = "上涨趋势"
    TRENDING_DOWN = "下跌趋势"
    RANGING = "震荡市"
    HIGH_VOLATILITY = "高波动"
```

**判断逻辑**:
```python
# 趋势市：ADX > 25 且 EMA10 > EMA20
TRENDING_UP: ADX > 25 AND EMA10 > EMA20 AND Price > EMA20

# 震荡市：ADX < 20
RANGING: ADX < 20

# 高波动：ATR > 1.5 × 平均 ATR
HIGH_VOLATILITY: ATR > 1.5 × Average(ATR, 20)
```

**动态阈值**:
| 市场状态 | 信号阈值 | 仓位系数 |
|----------|----------|----------|
| TRENDING_UP | 0.08% | 1.0x |
| TRENDING_DOWN | 0.08% | 1.0x |
| RANGING | 0.15% | 0.5x |
| HIGH_VOLATILITY | 0.20% | 0.3x |

**使用方法**:
```python
from v4_3.market_regime import MarketRegimeAgent

agent = MarketRegimeAgent()
regime = agent.detect_regime("EURUSD", "H1")

if agent.should_trade(regime):
    threshold = agent.get_dynamic_threshold(regime)
    print(f"可以交易，阈值：{threshold:.2f}%")
```

### 4.2 Factor Score Engine

**文件**: `v4_3/factor_score.py`

**功能**:
- 整合 4 大因子库
- 计算综合评分（0-100 分）
- 生成交易信号

**因子权重**:
```json
{
  "momentum": 0.3,
  "mean_reversion": 0.3,
  "breakout": 0.2,
  "volatility": 0.2
}
```

**评分计算**:
```python
总分 = Σ(因子_i 评分 × 权重_i)

# 转换到 0-100 范围
normalized_score = (原始分 + 100) / 2
```

**信号映射**:
| 评分 | 信号 | 含义 |
|------|------|------|
| ≥70 | STRONG_BUY | 强烈买入 |
| 55-69 | BUY | 买入 |
| 46-54 | NEUTRAL | 中性 |
| 31-45 | SELL | 卖出 |
| ≤30 | STRONG_SELL | 强烈卖出 |

**使用方法**:
```python
from v4_3.factor_score import FactorScoreEngine

engine = FactorScoreEngine()
result = engine.calculate_score("EURUSD", "H1")

print(f"评分：{result['score']:.1f}")
print(f"信号：{result['signal']}")
```

### 4.3 Risk Agent

**文件**: `v4_3/risk_agent.py`

**功能**:
- 账户级检查（保证金、杠杆）
- 仓位检查（总持仓、单一品种）
- VaR 检查（95% VaR ≤ 2% 净值）
- 相关性检查
- 实时否决权

**风控规则**:
```python
# 账户级
min_margin_level = 200%  # 保证金水平
max_leverage = 3.0x      # 实际杠杆

# 仓位级
max_positions = 3        # 总持仓数
max_same_symbol = 2      # 同一品种持仓
max_total_risk = 3%      # 总风险敞口

# 止损
stop_loss_atr = 1.5      # ATR 倍数
min_stop_points = 20     # 最小止损点数
```

**使用方法**:
```python
from v4_3.risk_agent import RiskAgent

agent = RiskAgent()

if agent.can_trade(symbol, signal, volume, account_info, positions):
    print("风控通过，可以交易")
else:
    print("风控不通过，禁止交易")
```

### 4.4 Review Agent

**文件**: `v4_3/review_agent.py`

**功能**:
- 策略归因（趋势/均值回归贡献）
- 品种归因（各币种盈亏）
- 时间归因（亚盘/欧盘/美盘）
- 因子归因（各因子 IC 变化）

agent = ReviewAgent(db_path="trading.db")

# 生成每日报告

# 模式识别
patterns = agent.identify_patterns(days=30)

## 五、因子库详解

### 5.1 动量因子

**文件**: `factors/momentum.py`

**子因子**:
1. **EMA 斜率**: (EMA10 - EMA20) / EMA20
2. **价格动量**: N 日涨幅
3. **MACD 信号**: MACD 线与信号线关系

**权重**: 30%

**适用场景**: 趋势市

---

### 5.2 均值回归因子

**文件**: `factors/mean_reversion.py`

**子因子**:
1. **RSI**: 超买超卖判断
2. **布林带位置**: 价格在布林带中的位置
3. **乖离率**: 价格与均线的偏离程度

**权重**: 30%

**适用场景**: 震荡市

---

### 5.3 突破因子

**文件**: `factors/breakout.py`

**子因子**:
1. **价格突破**: N 日高点/低点突破
2. **成交量**: 成交量放大
3. **形态识别**: 上升/下降/震荡形态

**权重**: 20%

**适用场景**: 突破行情

---

### 5.4 波动率因子

**文件**: `factors/volatility.py`

**子因子**:
1. **ATR 变化**: ATR 变化率
2. **布林带宽度**: 布林带收口/开口
3. **历史波动率**: 波动率百分位

**权重**: 20%

**适用场景**: 所有市场
