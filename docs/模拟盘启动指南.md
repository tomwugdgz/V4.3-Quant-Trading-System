
### 3.2 查看交易记录

**查询数据库**:
```bash
cd trading
sqlite3 paper_trading.db "SELECT * FROM trades ORDER BY created_at DESC LIMIT 10;"
```

**或使用 Python**:
```python
import sqlite3
import pandas as pd

conn = sqlite3.connect("trading/paper_trading.db")
df = pd.read_sql("SELECT * FROM trades ORDER BY created_at DESC LIMIT 10", conn)
print(df)
conn.close()
```

## 五、故障排除

### 5.1 MT5 连接失败

**错误**:
```
[ERROR] MT5 初始化失败
```

**解决**:
1. 启动 MetaTrader 5
2. 登录模拟账户
3. 确保数据连接正常
4. 重新启动脚本

---

### 5.2 配置文件不存在

**错误**:
```
[ERROR] 配置文件不存在：config/paper_trading_config.json
```

**解决**:
```bash
# 检查文件是否存在
dir config\paper_trading_config.json

# 如果不存在，创建默认配置
python v4_3/create_default_config.py
```

---

### 5.3 数据库错误

**错误**:
```
[ERROR] 数据库检查失败：no such table: trades
```

**解决**:
```bash
# 重新初始化数据库
python v4_3/init_database.py

# 或删除旧数据库
del trading\paper_trading.db
python v4_3\paper_trading_monitor.py
```
