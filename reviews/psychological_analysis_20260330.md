## 📈 心理模式识别

### 1. 情绪化交易模式

**触发条件**:
- 亏损后 → 急于翻本 → 报复性交易
- 盈利后 → 害怕失去 → 过早平仓

**典型案例**:
```
16:02 平仓 EURUSD BUY (亏损$29.75)
  ↓ (3 分钟后)
16:05 开仓 EURUSD SELL (报复性反向)
```

**心理机制**: 损失厌恶 (Loss Aversion) - 对亏损的痛苦感远大于盈利的快乐感

---

### 2. FOMO (害怕错过) 模式

**触发条件**:
- 看到"最强信号" → 怕错过 → 忽略质量阈值
- 价格快速波动 → 怕踏空 → 追涨杀跌

**典型案例**:
```
信号：SELL 0.056% (低于 0.1% 阈值)
  ↓ (FOMO 触发)
决策：BUY 0.17 手 (方向还搞反了！)
```

**心理机制**: 错失恐惧 (Fear Of Missing Out) - 害怕错过"机会"

---

### 3. 过度自信模式

**触发条件**:
- 连续盈利 → 自信膨胀 → 忽视规则
- 自认为"看透了" → 反向操作 → 违背信号

**典型案例**:
```
信号：SELL 0.056%
  ↓ (过度自信)
决策："我觉得应该 BUY" → 开 BUY
```

**心理机制**: 控制幻觉 (Illusion of Control) - 高估自己的判断能力

---
