## 📋 复盘流程

### Part 1: 今日交易回顾 (5 分钟)
```
1. 今日交易列表
2. 盈亏统计
3. 胜率/盈亏比
4. 最佳交易 / 最差交易
```

### Part 2: 市场分析 (10 分钟)
```
1. 全球新闻摘要 (调用 News API)
2. 重大事件影响
3. 货币对走势分析
4. 明日关注事件
```

### Part 3: Agent 点评 (15 分钟)
```
每位 Agent 发言:
- 高盛：基本面变化
- 大摩：技术面分析
- 桥水：风险评估
- 文艺复兴：量化信号
- 巴菲特：风险评价 ⭐
- 芒格：决策质量 ⭐
- 索罗斯：市场偏见 ⭐
```

### Part 4: 错误与改进 (10 分钟)
```
1. 今日错误分析
2. 根本原因
3. 改进措施
4. 记录到知识库
```

### Part 5: 明日计划 (5 分钟)
```
1. 关注货币对
2. 关键事件
3. 交易计划
4. 风险预算
```

### Part 6: 总结 (5 分钟)
```
1. 旺财总结
2. 飞书汇报
3. 保存记录
```

```json
{
  "name": "每日复盘会",
  "schedule": "0 21 * * *",
  "timezone": "Asia/Shanghai",
  "action": "daily_review",
  "participants": [
    "旺财",
    "高盛",

    "桥水",
    "摩根大通",
    "贝莱德",
    "城堡资本",
    "文艺复兴",
    "先锋",
    "D.E. 肖",
    "Two Sigma",
    "巴菲特",
    "查理芒格",
    "索罗斯"
  ]
}
```

### API 调用
```python
# 新闻 API
- NewsAPI: 全球新闻
- GNews: 备用新闻
- CurrentsAPI: 财经新闻
- MarketAux: 市场情绪
- Brave Search: 搜索补充

# 交易数据
- MT5 API: 交易记录
- Yahoo Finance: 行情数据
```

### 输出文件
```

trading/journal/YYYY-MM-DD_trade.md
memory/YYYY-MM-DD.md (更新)
```
