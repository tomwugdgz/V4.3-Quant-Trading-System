# 交易规则 (Trading Rules)

---

## ⚠️ 铁律 (不可违背)

### 0. 指令冲突提醒 (2026-03-26 新增)
- **当新指令同旧指令发生冲突时，必须 100% 提醒用户**
- **必须记入 memory 并显示提醒**
- **不得自行决定覆盖旧指令**
- **必须等用户确认后才执行新指令**

### 1. 风险管理
- **单笔风险**: 不超过账户的 2%
- **日最大亏损**: 不超过 5%
- **周最大亏损**: 不超过 10%
- **月最大亏损**: 不超过 15%
- **达到限额立即停止交易**

### 2. 止损纪律
- **每笔交易必须设置止损**
- **止损一旦设置，不得随意移动**
- **只能向有利方向移动止损 (追踪止损)**
- **止损被触发，立即离场，不抱幻想**

### 3. 仓位控制
- **不超过总资金的 20% 用于单笔交易**
- **同一方向不超过 50% 仓位**
- **相关品种不超过 30% 总仓位**

### 4. 交易时机
- **重大数据发布前后 30 分钟不交易**
- **流动性不足时不交易**
- **情绪不稳定时不交易**
- **生病/疲劳时不交易**

- [ ] 止盈单 (TP)

## 📊 持仓管理

### 移动止损规则
- **盈利达到 1R 后**: 移动止损到入场价 (保本)
- **盈利达到 2R 后**: 移动止损到 1R 利润处
- **盈利达到 3R 后**: 追踪止损，每新高/新低移动 1R

### 加仓规则
- **只在盈利头寸加仓**
- **每次加仓不超过初始仓位的 50%**
- **加仓后重新计算整体止损**
- **最多加仓 2 次**

### 减仓规则
- **达到第一目标位**: 减仓 50%
- **走势反转信号**: 减仓或平仓
- **重大事件前**: 减仓或平仓

## 🚫 禁止行为

### 绝对禁止
- ❌ 不止损/扛单
- ❌ 报复性交易
- ❌ 加仓摊平亏损
- ❌ 重仓赌方向
- ❌ 听消息交易
- ❌ 酒后/情绪化交易
- ❌ 频繁交易 (过度交易)
- ❌ 逆势加仓

### 需要警惕
- ⚠️ 连续盈利后过度自信
- ⚠️ 连续亏损后急于翻本
- ⚠️ 持仓过夜不设止损
- ⚠️ 同时开太多仓位
- ⚠️ 交易计划外品种

## 🎯 决策流程

### 开仓决策
```
1. 市场分析 (技术 + 基本面)
2. 识别交易机会
3. 计算风险回报比
4. 确定仓位大小
5. 设置止损止盈
6. 执行交易
7. 记录交易
```

### 平仓决策
```
1. 达到止盈 → 平仓或部分平仓
2. 触及止损 → 立即平仓
3. 走势反转 → 考虑平仓
4. 重大事件 → 考虑平仓
5. 需要资金 → 考虑平仓
```

## 📈 绩效目标

### 月度目标
- 胜率：≥ 45%
- 盈利因子：≥ 1.5
- 最大回撤：≤ 10%
- 月回报：≥ 2%

### 年度目标
- 胜率：≥ 50%
- 盈利因子：≥ 2.0
- 最大回撤：≤ 20%
- 年回报：≥ 20%

## 🔔 违规处罚

### 违反铁律
- **第一次**: 书面检讨，暂停交易 3 天
- **第二次**: 暂停交易 1 周，资金减半
- **第三次**: 暂停交易 1 个月，重新学习

### 违反一般规则
- **每次**: 记录在案，周末复盘时检讨
- **累计 3 次**: 暂停交易 1 天
