## 阶段性目标

### 第 1 年 (2026)
- **目标**: 生存 + 学习
- **重点**: 建立交易系统，控制风险
- **预期回报**: 不亏损就是胜利
- **关键指标**:
  - [ ] 建立完整的交易系统
  - [ ] 严格执行止损
  - [ ] 完成 100+ 笔交易
  - [ ] 胜率达到 45%+

### 第 2-3 年 (2027-2028)
- **目标**: 稳定盈利
- **重点**: 优化策略，提高胜率
- **预期回报**: 年化 20-30%
- **关键指标**:
  - [ ] 胜率 50%+
  - [ ] 盈利因子 1.5+
  - [ ] 最大回撤 <15%

### 第 4-5 年 (2029-2030)
- **目标**: 加速增长
- **重点**: 扩大仓位，多策略
- **预期回报**: 年化 40-50%
- **关键指标**:
  - [ ] 管理资金 ¥500,000+
  - [ ] 夏普比率 1.5+
  - [ ] 月均盈利 3%+

### 第 6-10 年 (2031-2035)
- **目标**: 复利增长
- **重点**: 保持纪律，持续复利
- **预期回报**: 年化 50-70%
- **关键指标**:
  - [ ] 管理资金 ¥5,000,000+
  - [ ] 最大回撤 <20%
  - [ ] 连续盈利月份 >24
