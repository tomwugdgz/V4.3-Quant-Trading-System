**算法类型**: 趋势跟踪  
**复杂度**: ⭐⭐⭐ (中)  
**计算成本**: ⭐⭐ (中)

#### 核心逻辑
```
使用 Kaufman Adaptive Moving Average
- 市场波动大时，AMA 反应快
- 市场波动小时，AMA 反应慢
- 自动适应市场状态

BUY:  价格上穿 AMA 且 AMA 向上
SELL: 价格下穿 AMA 且 AMA 向下
```

```json
{
  "fast_period": 2,
```

**算法类型**: 复合策略  
**复杂度**: ⭐⭐⭐⭐ (高)  
**计算成本**: ⭐⭐⭐ (高)

#### 核心逻辑
```
四重确认机制:
1. AMA (自适应均线) - 趋势方向
2. RSRS (阻力支撑相对强度) - 超买超卖 (>0.8)
3. ADX (平均趋向指数) - 趋势强度 (>25)
4. 标准差过滤 - 突破有效性

BUY:  四者全部确认多头
SELL: 四者全部确认空头
```

#### 参数配置
```json
{
  "ama_fast": 2,
  "ama_slow": 30,

  "adx_threshold": 25,

  "signal_threshold": 0.001,
```
