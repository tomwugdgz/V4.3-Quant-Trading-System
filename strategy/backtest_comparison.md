## 🎯 回测周期

### 短期回测 (7 天)
- **目的**: 快速验证新策略
- **数据量**: 约 10-30 单
- **决策**: 是否继续测试

### 中期回测 (30 天)
- **目的**: 月度绩效评估
- **数据量**: 约 40-120 单
- **决策**: 策略去留/优化

### 长期回测 (90 天)
- **目的**: 季度策略优化
- **数据量**: 约 120-360 单
- **决策**: 策略升级/淘汰

## 📈 对比维度

### 1. 盈利能力

| 指标 | 计算方法 | 重要性 |
|------|----------|--------|
| 总盈亏 | 所有交易盈亏总和 | ⭐⭐⭐⭐⭐ |
| 回报率 | 总盈亏 / 初始资金 | ⭐⭐⭐⭐⭐ |
| 日均收益 | 总盈亏 / 交易天数 | ⭐⭐⭐⭐ |

### 2. 风险控制

| 指标 | 计算方法 | 重要性 |
|------|----------|--------|
| 最大回撤 | 最大连续亏损 | ⭐⭐⭐⭐⭐ |
| 单日最大亏损 | 单日最大亏损额 | ⭐⭐⭐⭐⭐ |
| 波动率 | 日收益标准差 | ⭐⭐⭐⭐ |

### 3. 稳定性

| 指标 | 计算方法 | 重要性 |
|------|----------|--------|
| 胜率 | 盈利单数 / 总单数 | ⭐⭐⭐⭐⭐ |
| 盈利因子 | 总盈利 / 总亏损 | ⭐⭐⭐⭐⭐ |
| 夏普比率 | (收益 - 无风险) / 波动率 | ⭐⭐⭐⭐ |
| 卡玛比率 | 年化收益 / 最大回撤 | ⭐⭐⭐⭐ |

### 4. 交易质量

| 指标 | 计算方法 | 重要性 |
|------|----------|--------|
| 平均盈亏比 | 平均盈利 / 平均亏损 | ⭐⭐⭐⭐⭐ |
| 交易频率 | 交易数量 / 时间 | ⭐⭐⭐ |
| 平均持仓时间 | 总持仓时间 / 交易数量 | ⭐⭐⭐ |

## 🔧 回测执行流程

### Step 1: 数据准备

```python
# 获取历史数据
def prepare_backtest_data(strategy_version, start_date, end_date):
    """准备回测数据"""
    
    # 从数据库获取交易记录
    trades = get_trades_from_db(
        strategy=strategy_version,
        start=start_date,
        end=end_date
    )
    
    # 计算绩效指标
    stats = calculate_stats(trades)
    
    return stats
```

### Step 2: 指标计算

```python
def calculate_stats(trades):
    """计算策略绩效指标"""
    
    total_trades = len(trades)
    winning_trades = [t for t in trades if t['profit'] > 0]
    losing_trades = [t for t in trades if t['profit'] <= 0]
    
    win_rate = len(winning_trades) / total_trades if total_trades > 0 else 0
    
    total_profit = sum(t['profit'] for t in winning_trades)
    total_loss = abs(sum(t['profit'] for t in losing_trades))
    
    profit_factor = total_profit / total_loss if total_loss > 0 else float('inf')
    
    max_drawdown = calculate_max_drawdown(trades)
    
    sharpe_ratio = calculate_sharpe_ratio(trades)
    
    return {
        'total_trades': total_trades,
        'win_rate': win_rate,
        'total_profit': total_profit,
        'total_loss': total_loss,
        'profit_factor': profit_factor,
        'max_drawdown': max_drawdown,
        'sharpe_ratio': sharpe_ratio,
    }
```

### Step 3: 对比分析

```python
def compare_strategies(stats_list):
    """对比多个策略版本"""
    
    comparison = {
        'best_win_rate': max(stats_list, key=lambda x: x['win_rate']),
        'best_profit_factor': max(stats_list, key=lambda x: x['profit_factor']),
        'lowest_drawdown': min(stats_list, key=lambda x: x['max_drawdown']),
        'best_sharpe': max(stats_list, key=lambda x: x['sharpe_ratio']),
        'highest_return': max(stats_list, key=lambda x: x['total_profit']),
    }
    
    return comparison
```

### Step 4: 决策建议

```python
def make_decision(comparison, quality_gate):
    """根据对比结果和门禁系统做决策"""
    
    recommendations = []
    
    for strategy, stats in comparison.items():
        if stats['win_rate'] < quality_gate['min_win_rate']:
            recommendations.append(f"{strategy}: STOPPED (胜率过低)")
        elif stats['profit_factor'] < quality_gate['min_profit_factor']:
            recommendations.append(f"{strategy}: OPTIMIZING (盈利因子不足)")
        else:
            recommendations.append(f"{strategy}: ACTIVE (达标)")
    
    return recommendations
```
