## 一、趋势跟踪策略 (Trend Following) - 6 个

**核心逻辑**: 顺势而为，追涨杀跌  
**适用市场**: 强趋势、低震荡  
**代表指标**: MA、MACD、ADX

---

### TF-01: SMA 双均线交叉 (SMA Dual Crossover)

**专业名称**: Simple Moving Average Crossover  
**策略类型**: 趋势跟踪  
**复杂度**: ⭐ (入门级)

#### 核心逻辑
```
BUY 信号：SMA20 上穿 SMA50 (金叉)
SELL 信号：SMA20 下穿 SMA50 (死叉)
```

#### 参数
- 快线：20 周期
- 慢线：50 周期
- 止损：30 pips
- 止盈：60 pips

#### 预期表现
- 胜率：35-45%
- 盈亏比：1:2 至 1:3
- 频率：5-10 单/月

---

### TF-02: EMA 指数均线交叉 (EMA Exponential Crossover)

**专业名称**: Exponential Moving Average Crossover  
**策略类型**: 趋势跟踪  
**复杂度**: ⭐ (入门级)

#### 核心逻辑
```
BUY: EMA12 上穿 EMA26
SELL: EMA12 下穿 EMA26
```

#### 预期表现
- 胜率：40-50%
- 盈亏比：1:1.5 至 1:2
- 频率：10-15 单/月

---

### TF-03: MACD 趋势动能 (MACD Trend Momentum)

**专业名称**: Moving Average Convergence Divergence  
**策略类型**: 趋势跟踪 + 动量  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
BUY: MACD 金叉 + 柱状图由负转正
SELL: MACD 死叉 + 柱状图由正转负
```

#### 预期表现
- 胜率：45-55%
- 盈亏比：1:2
- 频率：8-12 单/月

---

### TF-04: ADX 趋势强度过滤 (ADX Trend Strength Filter)

**专业名称**: Average Directional Index  
**策略类型**: 趋势跟踪 + 过滤器  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
ADX > 25: 强趋势市 → 交易
ADX < 25: 震荡市 → 观望

+DI 上穿-DI + ADX>25: BUY
-DI 上穿+DI + ADX>25: SELL
```

#### 预期表现
- 胜率：50-60%
- 盈亏比：1:1.5 至 1:2
- 频率：5-8 单/月

---

### TF-05: 通道突破策略 (Donchian Channel Breakout)

**专业名称**: Donchian Channel Breakout  
**策略类型**: 趋势跟踪 + 突破  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
上轨 = N 周期最高价
下轨 = N 周期最低价

BUY: 价格突破上轨
SELL: 价格突破下轨
```

#### 预期表现
- 胜率：35-45%
- 盈亏比：1:3 至 1:4
- 频率：3-6 单/月

---

### TF-06: 自适应均线策略 (AMA Adaptive Moving Average)

**专业名称**: Kaufman Adaptive Moving Average  
**策略类型**: 趋势跟踪 + 自适应  
**复杂度**: ⭐⭐⭐ (专业级)

#### 核心逻辑
```
AMA 根据市场波动自动调整:
- 波动大 (趋势市): 反应快
- 波动小 (震荡市): 反应慢

BUY: 价格上穿 AMA 且 AMA 向上
SELL: 价格下穿 AMA 且 AMA 向下
```

#### 预期表现
- 胜率：50-60%
- 盈亏比：1:2
- 频率：8-12 单/月
- 夏普：1.2-1.5

## 二、均值回归策略 (Mean Reversion) - 5 个

**核心逻辑**: 价格偏离均值后必然回归  
**适用市场**: 震荡市、区间市

---

### MR-01: 布林带回归 (Bollinger Bands Mean Reversion)

**专业名称**: Bollinger Bands Mean Reversion  
**策略类型**: 均值回归  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
上轨 = SMA20 + 2 标准差
下轨 = SMA20 - 2 标准差

BUY: 价格触及下轨 + RSI<30
SELL: 价格触及上轨 + RSI>70
```

#### 预期表现
- 胜率：55-65%
- 盈亏比：1:1 至 1:1.5
- 频率：15-25 单/月

---

### MR-02: RSI 超买超卖反转 (RSI Overbought/Oversold)

**专业名称**: Relative Strength Index Reversal  
**策略类型**: 均值回归  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
RSI < 30: 超卖 → BUY
RSI > 70: 超买 → SELL
```

#### 预期表现
- 胜率：50-60%
- 盈亏比：1:1.5
- 频率：12-20 单/月

---

### MR-03: 随机指标反转 (Stochastic Oscillator)

**专业名称**: Stochastic Oscillator Reversal  
**策略类型**: 均值回归  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
%K 和%D < 20: 超卖
%K 上穿%D: 金叉 BUY

%K 和%D > 80: 超买
%K 下穿%D: 死叉 SELL
```

#### 预期表现
- 胜率：50-55%
- 盈亏比：1:1.5
- 频率：15-25 单/月

---

### MR-04: 威廉指标反转 (Williams %R)

**专业名称**: Williams Percent Range  
**策略类型**: 均值回归  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
%R < -80: 超卖 → BUY
%R > -20: 超买 → SELL
```

#### 预期表现
- 胜率：45-55%
- 盈亏比：1:1.5
- 频率：12-20 单/月

---

### MR-05: 肯特纳通道回归 (Keltner Channel)

**专业名称**: Keltner Channel Mean Reversion  
**策略类型**: 均值回归 + 波动率  
**复杂度**: ⭐⭐⭐ (专业级)

#### 核心逻辑
```
中轨 = EMA20
上轨 = 中轨 + ATR×2
下轨 = 中轨 - ATR×2

BUY: 价格跌破下轨后回归
SELL: 价格突破上轨后回归
```

#### 预期表现
- 胜率：55-65%
- 盈亏比：1:1.5 至 1:2
- 频率：10-18 单/月

## 三、动量策略 (Momentum) - 4 个

**核心逻辑**: 强者恒强，弱者恒弱  
**适用市场**: 突破市、趋势初期

---

### MO-01: ROC 动量突破 (Rate of Change)

**专业名称**: ROC Momentum Breakout  
**策略类型**: 动量突破  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
ROC = (当前价-N 周期前价)/N 周期前价

BUY: ROC 上穿零轴 + 成交量放大
SELL: ROC 下穿零轴 + 成交量放大
```

#### 预期表现
- 胜率：45-55%
- 盈亏比：1:2
- 频率：8-15 单/月

---

### MO-02: CCI 动量策略 (Commodity Channel Index)

**专业名称**: CCI Momentum  
**策略类型**: 动量 + 均值回归  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
CCI > +100: 趋势突破 BUY
CCI < -100: 趋势突破 SELL

CCI > +200: 超买回归 SELL
CCI < -200: 超卖回归 BUY
```

#### 预期表现
- 胜率：45-55%
- 盈亏比：1:1.5 至 1:2
- 频率：10-18 单/月

---

### MO-03: 价格动量突破 (Price Momentum)

**专业名称**: Price Momentum Breakout  
**策略类型**: 纯动量突破  
**复杂度**: ⭐⭐ (进阶级)

#### 核心逻辑
```
BUY: 价格突破 N 周期高点 + 动量>阈值
SELL: 价格突破 N 周期低点 + 动量<阈值
```

#### 预期表现
- 胜率：40-50%
- 盈亏比：1:2 至 1:3
- 频率：6-12 单/月

---

### MO-04: 量价动量策略 (Volume-Price Momentum)

**专业名称**: Volume-Price Momentum  
**策略类型**: 动量 + 成交量  
**复杂度**: ⭐⭐⭐ (专业级)

#### 核心逻辑
```
BUY: 价格上涨>1% + 成交量>均量×1.5
SELL: 价格下跌>1% + 成交量>均量×1.5
```

#### 预期表现
- 胜率：50-60%
- 盈亏比：1:2
- 频率：8-15 单/月

## 四、波动率策略 (Volatility) - 3 个

**核心逻辑**: 波动率收缩后必然扩张  
**适用市场**: 高波动、突破行情

---

### VO-01: ATR 波动率突破 (ATR Breakout)

**专业名称**: Average True Range Breakout  
**策略类型**: 波动率突破  
**复杂度**: ⭐⭐⭐ (专业级)

#### 核心逻辑
```
BUY: 价格突破前日高点+ATR×0.5
SELL: 价格跌破前日低点-ATR×0.5

止损：ATR×2
止盈：ATR×4
```

#### 预期表现
- 胜率：40-50%
- 盈亏比：1:2 至 1:3
- 频率：5-10 单/月

---

### VO-02: 标准差过滤 (Standard Deviation Filter)

**专业名称**: Standard Deviation Volatility Filter  
**策略类型**: 波动率过滤  
**复杂度**: ⭐⭐⭐ (专业级)

#### 核心逻辑
```
标准差 > 2: 高波动 → 交易
标准差 < 0.5: 低波动 → 观望
```

#### 预期表现
- 胜率：50-60%
- 盈亏比：1:2
- 频率：6-12 单/月

---

### VO-03: 波动率收缩突破 (Volatility Squeeze)

**专业名称**: Volatility Squeeze Breakout  
**策略类型**: 波动率收缩 + 突破  
**复杂度**: ⭐⭐⭐⭐ (高级)

#### 核心逻辑
```
阶段 1: 布林带宽度收缩至 N 周期最低 → 观望
阶段 2: 价格突破收缩区间 + 成交量放大 → 开仓
阶段 3: 持有到波动率再次收缩
```

#### 预期表现
- 胜率：50-60%
- 盈亏比：1:3 至 1:4
- 频率：3-6 单/月

## 📝 策略测试流程

### 第 1 步：回测验证
```
1. 至少 3 年历史数据
2. 包含不同市场状态 (趋势/震荡)
3. 考虑交易成本 (点差 + 佣金)
4. 样本外测试验证
```

### 第 2 步：模拟盘测试
