### v2.2 - 轻仓测试版 (2026-03-27)

**变更日期**: 2026-03-27  
**变更原因**: 增加交易频率，积累数据用于复盘

**核心变更**:
```
旧配置:
- 信号门槛：>0.1%
- 仓位大小：0.5% 风险 (约 0.17 手)
- 交易频率：低 (等高质量信号)

新配置:
- 信号门槛：>0.05% (降低 50%)
- 仓位大小：0.1-0.2% 风险 (约 0.03-0.05 手)
- 交易频率：中高 (增加数据积累)
```

**预期效果**:
- ✅ 交易数量增加 2-3 倍
- ✅ 一个月内积累 60+ 单数据
- ✅ 降低单笔风险，控制回撤

**风险**:
- ⚠️ 低质量信号可能增加亏损率
- ⚠️ 需要更严格监控

**测试期**: 2026-03-27 至 2026-04-27 (1 个月)

**决策人**: Tom  
**执行人**: 旺财 🎯

### v2.1 - 信号门槛优化 (2026-03-23)

**变更日期**: 2026-03-23  
**变更原因**: 提高信号质量，减少亏损交易

**核心变更**:
```
新增铁律:
1. 信号强度必须>0.1%
2. 平仓后等 30 分钟冷静期
3. 周末交易信号>0.2%
```

**实际效果**:
- ✅ 避免了低质量交易
- ❌ 交易频率过低，数据不足
- ❌ 无法有效复盘优化

**状态**: 🔴 STOPPED (2026-03-27)

**停止原因**: 交易频率太低，无法积累足够数据用于策略优化

### v2.0 - 四重确认策略 (2026-03-19)

**变更日期**: 2026-03-19  
**变更原因**: 提高信号准确率，减少假突破

**核心变更**:
```
引入四重确认:
1. AMA (自适应均线) - 趋势方向
2. RSRS (阻力支撑相对强度) - 超买超卖
3. ADX (>25) - 趋势强度
4. 标准差过滤 - 突破有效性
```

**实际效果**:
- ✅ 信号准确率提升
- ✅ 避免了多次假突破
- ⚠️ 有时过于保守，错过机会

**状态**: 🟡 OPTIMIZING

**优化方向**: 调整参数，平衡准确率与交易频率

### v1.0 - 初始策略 (2026-03-17)

**变更日期**: 2026-03-17  
**策略基础**: SMA20/SMA50 趋势跟踪

**核心逻辑**:
```
- 价格 > SMA20 > SMA50 → BUY
- 价格 < SMA20 < SMA50 → SELL
- 固定 0.01 手轻仓测试
```

**实际效果**:
- ✅ 简单直接，执行快
- ✅ 交易频率高，数据多
- ⚠️ 假突破较多，胜率一般

**状态**: 🟢 ACTIVE (作为基准策略)

## 🎯 策略质量门禁系统

### 状态定义

| 状态 | 标识 | 含义 | 动作 |
|------|------|------|------|
| **ACTIVE** | 🟢 | 策略健康，绩效达标 | 正常交易 |
| **OPTIMIZING** | 🟡 | 需要优化，绩效一般 | 暂停，回测调参 |
| **STOPPED** | 🔴 | 不合格，连续亏损 | 彻底停止 |
| **PENDING** | ⏳ | 新策略，测试中 | 轻仓测试 |

### 质量指标门禁

| 指标 | 达标标准 | 警告线 | 停止线 |
|------|----------|--------|--------|
| **胜率** | ≥45% | 35-45% | <35% |
| **盈利因子** | ≥1.5 | 1.0-1.5 | <1.0 |
| **最大回撤** | ≤10% | 10-15% | >15% |
| **夏普比率** | ≥1.0 | 0.5-1.0 | <0.5 |
| **交易数量** | ≥20 单/月 | 10-20 单 | <10 单 |
| **平均盈亏比** | ≥1:1.5 | 1:1-1:1.5 | <1:1 |

### 自动风控规则

```
IF 连续亏损 ≥ 5 单 THEN 状态 → OPTIMIZING
IF 单日亏损 ≥ 3% THEN 暂停交易 1 天
IF 单周亏损 ≥ 5% THEN 状态 → OPTIMIZING
IF 单月亏损 ≥ 10% THEN 状态 → STOPPED
IF 胜率 < 35% (20 单后) THEN 状态 → STOPPED
IF 盈利因子 < 1.0 (20 单后) THEN 状态 → STOPPED
```

## 📊 策略对比回测计划

### 回测周期
- **短期**: 7 天 (快速验证)
- **中期**: 30 天 (月度评估)
- **长期**: 90 天 (季度优化)

### 对比维度
1. **胜率**: 哪套策略更准确
2. **盈亏比**: 哪套策略赚得更多
3. **最大回撤**: 哪套策略更稳定
4. **夏普比率**: 哪套策略风险调整后收益更好
5. **交易频率**: 哪套策略更适合当前市场

### 回测数据源

- journal 交易记录
