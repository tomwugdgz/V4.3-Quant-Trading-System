### 示例 1: 2026-03-17 不止损的代价
**事件**: USDJPY 多单未设止损，从浮亏$2 扛到$XX
**损失**: $XX (如果及时止损只需亏$X)
**原因**: 
- 侥幸心理，认为会反弹
- 没有严格执行止损规则
- 情绪化决策

**教训**: 
- 止损是生存的保障，不是可选项
- 扛单只会让亏损扩大
- 必须在开仓时就设置止损

**改进措施**: 
- [x] 每笔交易强制设置止损
- [x] 止损设置后不随意移动
- [x] 定期回顾此教训

**状态**: ✅ 已建立规则

## 💡 重要认知

### 关于市场
1. 市场永远是对的
2. 不要试图预测，要跟随
3. 流动性为王
4. 趋势是你的朋友

### 关于风险
1. 生存第一，盈利第二
2. 风险永远放在第一位
3. 宁可错过，不要做错
4. 留得青山在，不怕没柴烧

### 关于自己
1. 承认自己的局限性
2. 情绪是最大的敌人
3. 纪律比聪明重要
4. 持续学习才能进步
