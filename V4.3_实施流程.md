### 阶段 2：Factor Score Engine（4/12-4/15）

**目标**: 解决"无稳定 Edge"问题

**交付物**:
- `factor_score.py` - 因子评分引擎
- `factors/` - 因子库
  - `momentum.py` - 动量因子
  - `mean_reversion.py` - 均值回归因子
  - `breakout.py` - 突破因子
- `factor_ic_analysis.py` - 因子 IC 分析

**评分逻辑**:
```python
总分 = Σ(因子_i * 权重_i)

因子库:
1. 动量因子 (权重 0.3)
   - EMA 斜率
   - 价格动量 (N 日涨幅)
   
2. 均值回归因子 (权重 0.3)
   - RSI 超买超卖
   - 布林带位置
   
3. 突破因子 (权重 0.2)
   - 突破 N 日高点/低点
   - 成交量放大
   
4. 波动率因子 (权重 0.2)
   - ATR 变化
   - 布林带宽度
```

**IC 分析**:
```python
IC = 因子值与未来收益的相关系数
IC > 0.05: 有效因子
IC < 0.02: 无效因子，剔除
```

### 阶段 3：Risk Agent 升级（4/15-4/17）

**目标**: 从"硬编码风控"升级为"动态风控"

**交付物**:
- `risk_agent.py` - 独立风控 Agent
- `var_calculator.py` - VaR 计算
- `exposure_monitor.py` - 敞口监控

**风控规则**:
```python
class RiskAgent:
    def can_trade(self, signal, account, positions):
        # 1. 账户级检查
        if account.margin_level < 200: return False
        if account.leverage > 3: return False
        
        # 2. 组合级检查
        total_risk = sum(pos.risk for pos in positions)
        if total_risk > 0.03: return False  # 总风险≤3%
        
        # 3. 相关性检查
        if self.check_correlation(signal.symbol, positions):
            return False  # 避免同向叠加
        
        # 4. VaR 检查
        var_95 = self.calculate_var(signal)
        if var_95 > account.equity * 0.02:
            return False  # 95% VaR ≤ 2% 净值
        
        return True
```

## 三、验收标准（Go/No-Go Gate）

### V4.3 验收指标

| 指标 | V4 标准 | 当前 v2.0 | V4.3 目标 |
|------|--------|----------|----------|
| 一致性 | ≥85 | 100% | ≥90 |
| 回撤控制 | ≥70 | 95% | ≥80 |
| 回测稳定性 | ≥70 | 45% | ≥70 |
| 胜率 | ≥55% | 52.8% | ≥58% |
| 盈亏比 | ≥1.5 | 1.9 | ≥1.8 |
| 综合评分 | ≥85 | 72.9 | ≥80 |

### 六大测试

1. **System Integrity Test** - 系统完整性测试
2. **Backtest Stress Test** - 回测压力测试
3. **Consistency Test** - 一致性测试（≥85）
4. **Risk Failure Test** - 风控失效测试
5. **Equity Stability Test** - 权益曲线稳定性
6. **Final Decision Gate** - 最终决策
