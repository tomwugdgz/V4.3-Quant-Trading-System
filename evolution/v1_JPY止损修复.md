## 变更内容

### 1. 止损参数（patrol.py - get_dynamic_sl_tp 函数）

**变更前 (v0)**:
```python
sl_pips = 15  # 固定
tp_pips = 30
```

**变更后 (v1)**:
```python
if is_jpy(symbol):
    sl_pips = max(atr_pips * 2.5, 25)   # JPY 最少 25 pips
else:
    sl_pips = max(atr_pips * 1.5, 15)   # 非 JPY 最少 15 pips
tp_pips = sl_pips * 2.0
```

### 2. JPY 专属交易规则

**变更前 (v0)**:
- JPY 与非 JPY 同样处理

**变更后 (v1)**:
- JPY 需额外满足: 波动率 > 0.25% 且 信号 > 80%
- 否则跳过，不交易

### 3. 信号算法
- 无变更（V3 信号算法保持不变）
- ADX + RSI + D1 趋势判断
