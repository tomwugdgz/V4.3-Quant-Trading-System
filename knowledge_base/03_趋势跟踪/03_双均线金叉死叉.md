## 📊 策略概述

**经典双均线交叉策略** - 最基础的趋势跟踪策略

**核心思路**: 
- 短期均线上穿长期均线 → 金叉 → 买入
- 短期均线下穿长期均线 → 死叉 → 卖出

**适用市场**: 股票、外汇、期货、加密货币

**时间框架**: 任意 (原策略日线，外汇建议 H4/D1)

---

## 🧮 策略逻辑

### 金叉 (买入信号)

```
条件：MA5 上穿 MA10

说明:
- 短期均线 (MA5) 从下方穿过长期均线 (MA10)
- 表明短期趋势转强
- 全仓买入 (原策略)
```

### 死叉 (卖出信号)

```
条件：MA5 下穿 MA10

说明:
- 短期均线 (MA5) 从上方穿过长期均线 (MA10)
- 表明短期趋势转弱
- 清仓出场
```

### 原策略参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 短期均线 | MA5 | 金叉/死叉判断 |
| 长期均线 | MA10 | 金叉/死叉判断 |
| 仓位管理 | 全仓 | 风险较大 |
| 止损 | 无 | 风险较大 |
| 时间框架 | D1 | 日线 |

---

## ⚠️ 原策略问题

### 1. 参数过于敏感

```
❌ MA5/MA10 在外汇市场噪音极大
✅ 建议：使用更长周期 (MA20/50 或 EMA21/55)
```

### 2. 无止损保护

```
❌ 没有止损 → 假信号会导致大幅亏损
✅ 建议：设置 ATR 止损或技术止损
```

### 3. 仓位管理激进

```
❌ 全仓进出 → 单笔风险过大
✅ 建议：单笔风险≤2%, 根据止损计算仓位
```

### 4. 无趋势过滤

```
❌ 震荡市会频繁金叉死叉，连续止损
✅ 建议：加入 ADX 或大周期均线过滤
```

## 🔧 策略改进方案

### 方案 1: 三重均线系统

```
均线组合：EMA10 + EMA21 + EMA55

入场 (多头):
1. EMA10 > EMA21 > EMA55 (多头排列)
2. EMA10 上穿 EMA21 (金叉确认)

出场:
1. EMA10 下穿 EMA21 (死叉)
2. 或价格 < EMA55 (趋势反转)

优点：过滤假信号，只在强趋势时交易
缺点：入场机会较少，滞后性增加
```

### 方案 2: 均线 +AD X 过滤

```
入场条件:
1. EMA21 上穿 EMA55 (金叉)
2. ADX(14) > 25 (趋势强度)
3. +DI > -DI (上涨方向)

出场条件:
1. EMA21 下穿 EMA55 (死叉)
2. 或 ADX < 20 (趋势减弱)

优点：避免震荡市假信号
缺点：ADX 滞后，可能错过部分行情
```

### 方案 3: 均线 +AMA 组合

```
趋势确认：AMA 向上
入场信号：EMA21 上穿 EMA55
出场信号：EMA21 下穿 EMA55 或 AMA 转向

优点：AMA 自适应，双重确认
缺点：复杂度增加
```

### 方案 4: 分批建仓

```
第一批：EMA21 上穿 EMA55, 仓位 30%
第二批：价格回调 EMA21 不破，仓位 30%
第三批：ADX > 25 确认，仓位 40%

出场:
- 第一批：EMA21 下穿 EMA55
- 第二批：价格 < EMA30
- 第三批：价格 < EMA40

优点：降低平均成本，减少假信号损失
缺点：管理复杂
```

## 📊 风险管理

### 仓位计算

```python
风险金额 = 账户余额 × 2%
止损距离 = 入场价 - 止损价 (点数)
点值 = 每点价值 (货币对决定)

仓位 = 风险金额 / (止损距离 × 点值)

示例:
账户：$10,000
风险：2% = $200
EURUSD 入场：1.1000
止损：1.0950 (50 点)
点值：$10/手

仓位 = 200 / (50 × 10) = 0.4 手
```

### 止损设置

```
固定止损：2-3×ATR
技术止损：前低/前高
时间止损：入场后 N 根 K 线不盈利则出场

止盈:
- 第一目标：2R (风险回报 1:2)
- 第二目标：移动止损
- 追踪止损：吊灯止损 (3×ATR)
```
