# 海龟交易策略 (Turtle Trading)

> 策略来源：聚宽 #49 | 学习时间：2026-03-18  
> 整理：旺财 | 费曼学习法笔记

---

## 1️⃣ 学习结论 (一句话)

**突破 N 日高点买入，跌破 N 日低点卖出，用 ATR 动态调整仓位和止损。**

---

## 2️⃣ 核心内容

### 策略逻辑

```python
入场信号:
- 价格突破 20 日高点 → 买入 1 个单位
- 价格继续上涨 0.5×ATR → 加仓 1 个单位
- 最多加仓 4 次 (共 4 个单位)

出场信号:
- 价格跌破 10 日低点 → 清仓 (多头)
- 价格跌破入场价 2×ATR → 止损出场

空头反向操作
```

### 核心参数

```
入场突破：20 日高点/低点
加仓间隔：0.5×ATR
最多加仓：4 次
止损：2×ATR
出场：10 日低点/高点

ATR 周期：20 日 (默认)
```

### 仓位计算 (核心！)

```python
单位仓位 = (账户总额 × 1%) / (ATR × 每点价值)

示例:
账户：$10,000
风险：1% = $100
ATR: 50 点
每点价值：$10

单位仓位 = 100 / (50 × 10) = 0.2 手

最多加仓 4 次 = 0.8 手总仓位
```

### 海龟交易法则精髓

```
1. 掌握优势 (Edge):
   - 突破交易，跟随趋势
   
2. 风险管理 (Risk):
   - 单笔风险 1%
   - 用 ATR 动态调整仓位
   
3. 一致性 (Consistency):
   - 严格执行规则
   - 不凭感觉交易
   
4. 简单 (Simplicity):
   - 规则简单明了
   - 机械化执行
```

## 3️⃣ 总结

### 优点
✅ 完整的交易系统 (入场 + 加仓 + 止损 + 出场)
✅ ATR 动态仓位管理 (适配波动率)
✅ 趋势跟踪，盈亏比高
✅ 规则明确，可机械化执行

### 缺点
❌ 震荡市频繁止损 (突破失败)
❌ 胜率较低 (40% 左右)
❌ 需要强大执行力 (人性挑战)
❌ 资金曲线波动大

```
- 初始止损：入场价 - 2×ATR
- 加仓后：最新入场价 - 2×ATR

- 价格跌破 10 日低点 → 全部清仓
```

### 与 AMA 结合

```
趋势过滤:
- AMA 向上 → 只做多突破
- AMA 向下 → 只做空突破
- AMA 走平 → 不交易 (震荡市)

这样可以过滤掉部分震荡市的假突破！
```
