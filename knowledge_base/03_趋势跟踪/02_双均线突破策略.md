## 📊 策略概述

**最简单的趋势跟踪策略** - 均线突破入场，均线跌破出场

**核心思路**: 价格突破短期均线入场，跌破长期均线出场

**适用市场**: 股票、外汇、期货、加密货币

**时间框架**: 任意 (原策略为日线，外汇建议 H4/D1)

---

## 🎯 原策略逻辑

### 买入条件

```
当前价格 > 1.01 × MA10

说明:
- 价格突破 10 日均线
- 1% 的突破确认 (过滤假突破)
- 全仓买入 (原策略)
```

### 卖出条件

```
当前价格 < MA5

说明:
- 价格跌破 5 日均线
- 清仓出场
```

### 原策略参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 短期均线 | MA5 | 出场信号 |
| 长期均线 | MA10 | 入场信号 |
| 突破确认 | 1% | 过滤假突破 |
| 仓位管理 | 全仓 | 风险较大 |

---

## ⚠️ 原策略问题

### 1. 仓位管理过于激进

```
❌ 全仓进出 → 单笔风险过大
✅ 建议：单笔风险≤2%, 根据止损距离计算仓位
```

### 2. 无止损保护

```
❌ 没有止损 → 可能遭遇大幅回撤
✅ 建议：设置 ATR 止损或技术止损
```

### 3. 参数过于简单

```
❌ MA5/MA10 在外汇市场噪音大
✅ 建议：使用更长周期 (MA20/MA50)
```

### 4. 无趋势过滤

```
❌ 震荡市会频繁止损
✅ 建议：加入 ADX 或大周期均线过滤
```

## 🔧 策略改进方案

### 方案 1: 三重均线系统

```
入场：价格 > EMA15 > EMA21 > EMA50 (多头排列)
出场：价格 < EMA15 或 EMA15 < EMA21

优点：过滤震荡，只在强趋势时交易
缺点：入场机会较少
```

### 方案 2: 均线 +AD X 过滤

```
入场条件:
1. 价格 > EMA21
2. ADX(14) > 25 (趋势强度)
3. +DI > -DI (上涨方向)

出场条件:
1. 价格 < EMA50
2. 或 ADX < 20 (趋势减弱)

优点：避免震荡市
缺点：ADX 滞后
```

### 方案 3: 均线 +AMA 组合

```
趋势确认：AMA 向上
入场信号：价格 > EMA21
出场信号：价格 < EMA15 或 AMA 转向

优点：AMA 自适应，AMA+ 均线双重确认
缺点：复杂度增加
```

### 方案 4: 分批建仓

```
第一批：价格 > EMA21, 仓位 30%
第二批：价格 > EMA21 + ADX > 25, 仓位 30%
第三批：价格回调 EMA21 不破，仓位 40%

出场:
- 第一批：价格 < EMA30
- 第二批：价格 < EMA40
- 第三批：价格 < EMA50

优点：降低平均成本
缺点：管理复杂
```

## 📊 风险管理

### 仓位计算

```python
# 基于风险的仓位计算
风险金额 = 账户余额 × 2%
止损距离 = 入场价 - 止损价 (点数)
点值 = 每点价值 (货币对决定)

仓位 = 风险金额 / (止损距离 × 点值)

示例:
账户：$10,000
风险：2% = $200
EURUSD 入场：1.1000
止损：1.0950 (50 点)
点值：$10/手

仓位 = 200 / (50 × 10) = 0.4 手
```

### 止损设置

```
固定止损：2-3×ATR
技术止损：前低/前高
时间止损：入场后 N 根 K 线不盈利则出场

止盈:
- 第一目标：2R (风险回报 1:2)
- 第二目标：移动止损
- 追踪止损：吊灯止损
```
