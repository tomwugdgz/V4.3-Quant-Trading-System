# 配对交易策略 (Pairs Trading)

> 策略来源：聚宽 #50, #55 | 学习时间：2026-03-18

---

## 📊 策略概述

**核心思路**: 找两个高度相关的品种，当价差偏离正常范围时，做空强的、做多弱的，等待回归

**适用市场**: 股票、外汇、期货

**策略类型**: 市场中性、均值回归

---

## 🧮 配对交易原理

### 协整关系

```
两个品种价格序列 P1 和 P2

如果存在协整关系:
P2 - β × P1 = 平稳序列

β = 回归系数 (对冲比率)
```

### Z-Score 计算

```python
# 价差 = P2 - β × P1
spread = price2 - hedge_ratio * price1

# 计算 N 日均值和标准差
mean = spread.rolling(N).mean()
std = spread.rolling(N).std()

# Z-Score = (当前价差 - 均值) / 标准差
z_score = (spread - mean) / std
```

### 交易信号

| Z-Score | 信号 | 操作 |
|---------|------|------|
| > 1.5 | 价差过高 | 做空 P2 + 做多 P1 |
| < -1.5 | 价差过低 | 做多 P2 + 做空 P1 |
| 回归 0 | 平仓 | 两边同时平仓 |

---

## 🎯 交易规则

### 入场条件

```
做多价差 (Long Spread):
1. Z-Score < -1.5
2. 价差开始回升
3. 相关性 > 0.7 (过去 60 日)

做空价差 (Short Spread):
1. Z-Score > 1.5
2. 价差开始回落
3. 相关性 > 0.7
```

### 仓位配置

```
对冲比率 β = 回归系数

做多腿仓位 = 基础仓位
做空腿仓位 = 基础仓位 × β

目标：市场中性，对冲方向性风险
```

### 出场条件

```
止盈:
1. Z-Score 回归 0 (完全回归)
2. Z-Score 回归到 0.5 (部分止盈)
3. 固定止盈：5-10%

止损:
1. Z-Score 继续扩大到 2.5
2. 相关性跌破 0.5
3. 固定止损：5%
```

## ⚠️ 注意事项

### 配对交易的风险

1. **协整关系破裂**: 原本相关的品种突然脱钩
   - 原因：政策变化、经济事件
   - 解决：持续监控相关性，跌破阈值平仓

2. **单边行情**: 市场出现强烈趋势
   - 表现：价差持续扩大，不回归
   - 解决：设置止损，不扛单

3. **流动性风险**: 一边流动性差
   - 表现：点差扩大，滑点增加
   - 解决：选择流动性好的品种

### 最佳使用场景

| 场景 | 适用性 | 说明 |
|------|--------|------|
| 震荡市 | ⭐⭐⭐⭐⭐ | 价差频繁回归 |
| 温和趋势 | ⭐⭐⭐ | 价差缓慢变化 |
| 剧烈趋势 | ⭐ | 价差持续扩大，易止损 |
| 重大事件 | ⚠️ 避免 | 协整关系可能破裂 |

## 🔧 策略改进方向

### 改进 1: 动态对冲比率

```
每周/每月重新计算β
→ 适应关系变化
```

### 改进 2: 多配对分散

```
同时交易 3-5 个配对
→ 分散风险，平滑收益
```

### 改进 3: 趋势过滤

```
ADX > 30 时暂停配对交易
→ 避免趋势市连续止损
```
