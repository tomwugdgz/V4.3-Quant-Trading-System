# RSRS 择时策略 (阻力支撑相对强度)

> 策略来源：聚宽 #69, #77 | 学习时间：2026-03-18

---

## 📊 策略概述

**核心思路**: 用最高价对最低价做线性回归，斜率表示阻力/支撑的相对强度

**适用市场**: 股票、外汇、期货

**时间框架**: D1/H4 (外汇)

---

## 🧮 RSRS 原理

### 计算方法

```python
# 取 N 日的 High 和 Low 数据
highs = [H1, H2, ..., HN]
lows = [L1, L2, ..., LN]

# 用 High 对 Low 做线性回归
# High = α + β × Low + ε

# RSRS 斜率 = β (回归系数)
# RSRS 截距 = α
```

### 标准化 RSRS

```
RSRS 标准分 = (当前β - N 日β均值) / N 日β标准差

右偏优化:
RSRS_right = RSRS 标准分 × β × R²
```

### 交易信号

| RSRS 标准分 | 信号 | 含义 |
|-------------|------|------|
| > 0.7 | 做多 | 阻力强于支撑，上涨概率大 |
| < -0.7 | 做空 | 支撑强于阻力，下跌概率大 |
| -0.7 ~ 0.7 | 观望 | 震荡市，不交易 |

## 🎯 交易规则

### 入场条件 (做多)

```
1. RSRS 标准分 > 0.7
2. (可选) RSRS_right > 0.9 (右偏优化)
3. (可选) 价格 > MA20 (趋势确认)
4. (可选) ADX > 25 (趋势强度)

→ 同时满足才入场
```

### 入场条件 (做空)

```
1. RSRS 标准分 < -0.7
2. (可选) RSRS_right < -0.9
3. (可选) 价格 < MA20
4. (可选) ADX > 25

→ 同时满足才入场
```

### 出场条件

```
止损:
- 固定止损：2-3×ATR
- 技术止损：前低/前高

止盈:
- RSRS 标准分回归 0
- RSRS 反向信号
- 追踪止损：吊灯止损
```

## ⚠️ 注意事项

### RSRS 的优点

1. **前瞻性**: 比传统指标更早发出信号
2. **自适应**: 自动适应不同波动率
3. **简单**: 计算简单，逻辑清晰

### RSRS 的缺点

1. **震荡市失效**: 横盘时频繁假信号
   - 解决：用 ADX 过滤

2. **滞后性**: 需要 N 日数据才能计算
   - 解决：缩短周期但增加噪音

3. **极端行情**: 暴涨暴跌时信号滞后
   - 解决：配合止损

## 🔧 策略改进方向

### 改进 1: RSRS + AMA

```
RSRS 判断方向
AMA 确认趋势
→ 双重确认，减少假信号
```

### 改进 2: 多周期共振

```
D1 RSRS > 0.7 (主趋势向上)
H4 RSRS > 0.7 (次级趋势向上)
H4 价格回踩 AMA → 入场
```

### 改进 3: 右偏优化

```
RSRS_right = RSRS 标准分 × β × R²

优势: 考虑拟合优度，信号更可靠
劣势: 计算复杂，参数多
```
