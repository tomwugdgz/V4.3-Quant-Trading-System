### 三重过滤器
```
1. 标准差过滤 (FilterTimes=0.3-0.5)
   - 识别震荡市
   - 来源：AdaptiveMA 策略

2. ADX 趋势强度 (>25)
   - 确认趋势强度
   - 来源：DMI 策略 (#79)

3. RSRS 加权指标 (>0.8)
   - 高低点回归确认
   - 来源：RSRS 策略 (#27, #77)
```

### 四重入场确认
```
1. AMA 方向确认 (主信号)
2. 三重过滤器全部通过
3. KDJ/MACD/RSI 任一超卖/超买
4. 价格突破关键位 (前高/前低)
```

### 分批建仓 (海龟启发)
```
第一批：30% 仓位 (初始信号)
第二批：30% 仓位 (回调确认)
第三批：40% 仓位 (ADX 确认)

最多加仓 3 次，避免过度暴露
```

### 止损系统 (海龟 + 吊灯)
```
初始止损：2.5-3×ATR
追踪止损：吊灯止损 (3×ATR)
时间止损：20 根 K 线不盈利

止损后 20 根 K 线冷却 (防重入)
```

### 风险管理
```
单笔风险：≤2%
每日止损：≤5%
每周止损：≤10%
每月止损：≤15%

最大持仓：2-3 个货币对
相关性控制：避免高相关货币对
```

### 出场策略
```
1. 止损触发 → 立即出场
2. 止盈触发 → 部分出场 (50%)
3. 信号反转 → 全部出场
4. 时间止损 → 保本出场
```
