## 🎯 批量策略核心要点

### 1️⃣ 协整配对交易 (#02)

**结论**: 两个相关股票价格偏离时做空高价股、做多低价股，等待回归。

**核心**:
- 选择协整股票对 (如伊利股份 + 招商银行)
- 计算价格比率
- Z-Score > 2: 做空高价股，做多低价股
- Z-Score < -2: 反向操作
- Z-Score 回归 0: 平仓

**外汇应用**: EURUSD/GBPUSD、AUDUSD/NZDUSD 配对

---

### 2️⃣ SVR 机器学习策略 (#03, #06)

**结论**: 用支持向量机预测股票收益，选前 10 名买入。

**核心**:
- 因子：市值、负债率、盈利增长、研发投入
- SVR 模型预测
- 每 10 天调仓
- 持有 10 只股票

**外汇应用**: 可用技术指标替代基本面因子训练模型

---

### 3️⃣ 商品期货效应 (#04)

**结论**: 商品期货价格变动对相关股票有预测作用。

**核心**:
- 原油价格 → 石油股
- 铜价 → 铜业股
- 期货领涨 → 买入相关股票

**外汇应用**: 美元指数 → 货币对强弱

---

### 4️⃣ 期货团队持仓分析 (#05)

**结论**: 跟踪期货公司持仓，跟随主力方向。

**核心**:
- 分析期货公司多空持仓
- 跟随净多头/净空头
- 定期调仓

**外汇应用**: CFTC 持仓报告 (COT)

---

### 5️⃣ 股票估值逻辑 (#06)

**结论**: 股票估值是当下价格，不是历史也不是未来。

**核心**:
- PE/PB 要与当前盈利匹配
- 不要过度依赖预测
- 关注当前财务数据

**外汇应用**: 不适用 (外汇无估值概念)

---

### 6️⃣ 券商金股策略 (#07)

**结论**: 跟踪券商推荐的金股组合。

**核心**:
- 收集券商月度金股
- 构建组合
- 定期调仓

**外汇应用**: 不适用

---

### 7️⃣ 动量轮动策略 (#10)

**结论**: 买入近期强势股，卖出弱势股。

**核心**:
- 计算 N 日收益率
- 买入前 N 名
- 定期轮动

**外汇应用**: 货币对强弱轮动

---

### 8️⃣ 港股通策略 (#11)

**结论**: 跟踪港股通资金流向。

**核心**:
- 北向资金流入 → 买入
- 北向资金流出 → 卖出

**外汇应用**: 资金流向分析

---

### 9️⃣ 小市值策略 (#16)

**结论**: 买入小市值股票，长期跑赢大盘。

**核心**:
- 选择市值最小的一批股票
- 定期调仓
- 长期持有

**外汇应用**: 不适用

---

### 🔟 ATR 自适应通道 (#20)

**结论**: 用 ATR 动态调整通道宽度，突破交易。

**核心**:
- 中轨：MA(N)
- 上轨：中轨 + ATR×倍数
- 下轨：中轨 - ATR×倍数
- 突破上轨做多，跌破下轨做空

**外汇应用**: ⭐⭐⭐⭐⭐ 直接可用

## 💡 核心模式识别

## 🎯 策略 2.0 最终融合方案

基于今晚学习的 45+ 个策略，策略 2.0 最终方案：

### 核心信号
```
主信号：AMA 自适应均线 (KAMA, window=15)
```

### 三重过滤器
```
1. 标准差过滤 (FilterTimes=0.3-0.5)
   - 识别震荡市
   
2. ADX 趋势强度 (>25)
   - 确认趋势强度
   
3. RSRS 加权指标 (>0.8)
   - 高低点回归确认
```

### 四重入场确认
```
1. AMA 方向确认
2. 过滤器全部通过
3. KDJ/MACD/RSI 任一超卖/超买
4. 价格突破关键位
```

### 分批建仓
```
第一批：30% 仓位 (初始信号)
第二批：30% 仓位 (回调确认)
第三批：40% 仓位 (ADX 确认)
```

### 止损系统
```
初始止损：2.5-3×ATR
追踪止损：吊灯止损 (3×ATR)
时间止损：20 根 K 线不盈利
```

### 防重入机制
```
止损后 20 根 K 线冷却
避免重复止损
```

### 风险管理
```
单笔风险：≤2%
每日止损：≤5%
最大持仓：2-3 个货币对
相关性控制：避免高相关货币对
```
