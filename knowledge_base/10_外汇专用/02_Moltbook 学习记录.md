## 🎯 1000 倍杠杆交易要点

### 用户配置

```
杠杆：1000 倍
品种：外汇 (USD/JPY/CNY 等)
风格：超短线/短线
```

### 核心风险

| 杠杆 | 爆仓所需反向波动 | 风险等级 |
|------|------------------|----------|
| 100 倍 | 1% | ⭐⭐⭐ |
| 500 倍 | 0.2% | ⭐⭐⭐⭐ |
| 1000 倍 | 0.1% | ⚠️ 极高危 |

### 生存法则

```
1. 仓位必须极小 (0.01 手起步)
2. 止损必须极紧 (3-8 点)
3. 止盈必须快速 (8-15 点)
4. 绝不允许扛单
5. 每日止损上限 3%
6. 盈利立即出金 50%
```

## 📝 1000 倍杠杆特别建议

### 仓位计算公式

```
仓位 (手) = (账户余额 × 风险%) / (止损点数 × 每点价值)

示例:
账户：$10,000
风险：0.5% = $50
止损：5 点
EURUSD 每点价值：$10 (标准手)

仓位 = 50 / (5 × 10) = 1.0 手

但 1000 倍杠杆下:
- 1 手需要保证金：约 $100
- 可用保证金：$10,000
- 建议：从 0.1 手开始练习
```

### 每日交易计划

```
最大交易笔数：10 笔/天
单笔风险：0.5%
每日止损上限：3% (6 笔连续亏损)
每日盈利目标：5% (达到可停止)

交易时段:
- 最佳：20:00-23:00 (美欧重叠)
- 避免：0:00-8:00 (流动性低)
```

### 必设保护

```
1. 硬止损 (Hard Stop): 每笔必设
2. 每日止损限额：达到自动停止
3. 周末不平仓：周五必须清仓
4. 新闻不交易：重大数据前后 30 分钟避免
```
