## 📊 超短线交易概述

**定义**: 持仓时间从几秒到几分钟，赚取小幅价格波动

**特点**:
- 高频交易 (每天 10-50 笔)
- 小止盈 (5-15 点)
- 紧止损 (3-10 点)
- 高胜率 (目标 65%+)
- 快进快出

**适用杠杆**: 500-1000 倍 (用户配置)

---

## ⚠️ 1000 倍杠杆的风险警示

### 杠杆计算

```
账户：$10,000
杠杆：1000 倍
可用保证金：$10,000,000

标准手 (100,000 单位):
- EURUSD 波动 1 点 = $10
- 反向 100 点 = 爆仓 (1% 波动)

微型手 (1,000 单位):
- EURUSD 波动 1 点 = $0.10
- 反向 1000 点 = 爆仓 (10% 波动)
```

### 风险管理 (关键！)

| 风险等级 | 单笔风险 | 仓位计算 | 建议 |
|----------|----------|----------|------|
| 保守 | 0.5% | $50 / (止损点数×点值) | ⭐⭐⭐⭐⭐ |
| 中等 | 1% | $100 / (止损点数×点值) | ⭐⭐⭐⭐ |
| 激进 | 2% | $200 / (止损点数×点值) | ⭐⭐⭐ |
| 危险 | 5%+ | $500+ / (止损点数×点值) | ⚠️ 避免 |

### 1000 倍杠杆的生存法则

```
1. 仓位必须小 (0.01-0.1 手起步)
2. 止损必须紧 (5-10 点)
3. 止盈必须快 (10-20 点)
4. 绝不扛单
5. 每日止损上限 (3-5%)
6. 盈利后出金 (保护利润)
```

## 🎯 超短线核心技术

### 1. 价格行为 (Price Action)

```
关键概念:
- 支撑/阻力位
- 趋势线
- 通道
- K 线形态 (Pin Bar、吞没、Inside Bar)

交易逻辑:
价格触及支撑 → 观察 K 线反应 → 信号确认 → 入场
```

### 2. 订单流 (Order Flow)

```
观察:
- 大单方向
- 买卖压力
- 流动性区域

工具:
- 深度图 (DOM)
- 成交量分布
- 足迹图 (Footprint)
```

### 3. 市场结构

```
趋势结构:
上涨：Higher High + Higher Low
下跌：Lower High + Lower Low

突破结构:
- 区间突破
- 趋势线突破
- 前高/前低突破
```

## 📈 超短线策略

### 策略 1: 剥头皮 (Scalp)

```
时间框架：M1/M5
止盈：5-10 点
止损：3-5 点
胜率目标：70%+

入场:
- 价格触及支撑/阻力
- K 线反转信号
- 快速入场，快速出场
```

### 策略 2: 突破交易

```
时间框架：M5/M15
止盈：15-30 点
止损：8-12 点
胜率目标：55%+

入场:
- 区间突破
- 新闻事件突破
- 关键位突破
```

### 策略 3: 回调交易

```
时间框架：M15/H1
止盈：20-50 点
止损：10-15 点
胜率目标：60%+

入场:
- 趋势中的回调
- 回调至支撑/阻力
- 趋势恢复信号
```

## ⚠️ 超短线的风险

### 主要风险

| 风险 | 说明 | 应对 |
|------|------|------|
| 滑点 | 快速行情下成交价差 | 避免新闻时段 |
| 点差扩大 | 流动性低时点差增加 | 选择流动性好的品种 |
| 延迟 | 网络/平台延迟 | 选择优质经纪商 |
| 过度交易 | 频繁交易累积亏损 | 设置每日交易上限 |
| 情绪失控 | 连续亏损后报复性交易 | 严格执行止损 |

### 1000 倍杠杆的特别警示

```
⚠️ 1000 倍杠杆下:
- 1% 反向波动 = 100% 损失
- 必须用最小仓位 (0.01 手)
- 必须设置紧止损 (5-10 点)
- 必须严格执行纪律
- 建议先用模拟账户练习
```

## ⚠️ 重要提醒

### 1000 倍杠杆生存建议

1. **先用模拟账户练习 1 个月**
   - 熟悉超短线节奏
   - 验证策略胜率
   - 建立交易纪律

2. **实盘从小仓位开始**
   - 第一周：0.01 手
   - 第二周：0.02 手
   - 稳定后再增加

3. **严格止损**
   - 每笔必设止损
   - 止损 5-10 点
   - 触及立即平仓

4. **每日止损上限**
   - 亏损达 3% → 停止交易
   - 亏损达 5% → 强制休息 1 天

5. **盈利出金**
   - 每周出金 50% 利润
   - 保护既得收益
   - 避免回吐
