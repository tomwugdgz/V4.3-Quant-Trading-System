# 布林带策略 (Bollinger Bands)

> 策略来源：聚宽 #45, #52 | 学习时间：2026-03-18

---

## 📊 策略概述

**核心思路**: 利用布林带的收口和开口判断波动率变化，配合价格位置交易

**适用市场**: 股票、外汇、期货、加密货币

**时间框架**: H4/D1 (外汇)

---

## 🧮 布林带原理

```
中轨 = N 日简单移动平均线 (SMA)
上轨 = 中轨 + K × N 日标准差
下轨 = 中轨 - K × N 日标准差

默认参数：N=20, K=2
```

### 核心概念

| 现象 | 含义 | 交易信号 |
|------|------|----------|
| 收口 (Squeeze) | 波动率降低 | 即将突破，准备入场 |
| 开口 (Expansion) | 波动率增加 | 趋势确立，跟随方向 |
| 触上轨 | 超买 | 可能回调或继续突破 |
| 触下轨 | 超卖 | 可能反弹或继续下跌 |
| 沿上轨运行 | 强势上涨 | 持有多单 |
| 沿下轨运行 | 强势下跌 | 持有空单 |

## 🎯 交易策略

### 策略 1: 突破交易 (趋势市)

```
入场条件:
1. 布林带收口后突然开口
2. 价格突破上轨 → 做多
3. 价格突破下轨 → 做空
4. 成交量放大确认

止损:
- 突破 K 线的低点 (做多)
- 突破 K 线的高点 (做空)

止盈:
- 第一目标：2R
- 追踪止损：中轨
```

### 策略 2: 均值回归 (震荡市)

```
入场条件:
1. 布林带水平运行 (无明显趋势)
2. 价格触上轨 + KDJ 超买 → 做空
3. 价格触下轨 + KDJ 超卖 → 做多
4. 价格回到中轨确认

止损:
- 上轨上方 (做空)
- 下轨下方 (做多)

止盈:
- 中轨 (第一目标)
- 对面轨道 (第二目标)
```

### 策略 3: 布林带挤压 (Squeeze)

```
入场条件:
1. 布林带宽度创 N 日新低 (极度收口)
2. 等待方向突破
3. 突破上轨 → 做多
4. 突破下轨 → 做空

优势: 捕捉大趋势起点
劣势: 假突破较多，需要过滤
```

## 💡 外汇应用改进

### 参数调整

| 参数 | 默认 | 外汇 H4 | 外汇 D1 |
|------|------|---------|---------|
| 周期 N | 20 | 20 | 20 |
| 标准差 K | 2 | 2.5 | 2 |
| 均线类型 | SMA | EMA | SMA |

### 配合指标

| 指标 | 作用 | 参数 |
|------|------|------|
| ADX | 判断趋势强度 | 14, 阈值 25 |
| KDJ | 超买超卖确认 | 14, 3, 3 |
| MACD | 趋势方向确认 | 12, 26, 9 |
| ATR | 波动率确认 | 14 |

| 货币对 | 波动率 | 建议参数 |

## ⚠️ 注意事项

### 常见陷阱

1. **假突破**: 收口后突破但迅速回调
   - 解决：等收盘价确认 + 成交量

2. **趋势市做反转**: 强势趋势中触轨就反向
   - 解决：用 ADX 判断趋势强度

3. **参数固定不变**: 不同品种用同一参数
   - 解决：根据波动率调整 K 值

### 最佳使用场景

| 场景 | 策略选择 | 成功率 |
|------|----------|--------|
| 低波动收口 | 突破交易 | ⭐⭐⭐⭐ |
| 高波动开口 | 趋势跟随 | ⭐⭐⭐⭐ |
| 水平震荡 | 均值回归 | ⭐⭐⭐ |
| 强势单边 | 沿轨持有 | ⭐⭐⭐⭐⭐ |

## 🔧 策略改进方向

### 改进 1: 布林带 +AMA

```
用 AMA 替代 SMA 作为中轨
→ 自适应调整，减少滞后
```

### 改进 2: 动态带宽

```
根据 ATR 调整带宽
高波动：K=2.5
低波动：K=1.8
```

### 改进 3: 多周期共振

```
D1: 确定主趋势方向
H4: 布林带信号
H1: 精确入场
```
