# 布林带 (Bollinger Bands) 策略

> 策略来源：聚宽 #45, #52 | 学习时间：2026-03-18  
> 整理：旺财 | 费曼学习法笔记

---

## 1️⃣ 学习结论 (一句话)

**价格跌破下轨买入 (超卖)，突破上轨卖出 (超买)，均值回归策略。**

---

## 2️⃣ 核心内容

### 策略逻辑

```python
版本 1 (#45):
入场：价格 > 上轨 → 做空 (价格过高，等待回调)
出场：价格 < 下轨 → 平仓

版本 2 (#52):
入场：价格 < 下轨 → 做多 (价格过低，等待反弹)
出场：价格 > 上轨 → 平仓

注意：两个策略逻辑相反，需根据市场选择
```

### 布林带参数

```
周期：20 (默认)
标准差倍数：2 (上下轨)

计算:
中轨 = MA(20)
上轨 = 中轨 + 2×标准差
下轨 = 中轨 - 2×标准差
```

### 布林带原理

```
上轨 (阻力):
- 价格 > 上轨：超买
- 可能回调

下轨 (支撑):
- 价格 < 下轨：超卖
- 可能反弹

中轨 (均值):
- 价格趋向中轨
- 突破中轨表示趋势变化

带宽 (Bandwidth):
- 带宽收窄：波动率低，即将突破
- 带宽扩张：波动率高，趋势延续
```

---

## 3️⃣ 总结

### 优点
✅ 直观显示超买超卖
✅ 自适应波动率 (标准差)
✅ 适合震荡市
✅ 可判断突破 (带宽收窄)

### 缺点
❌ 强趋势市失效 (价格可沿上轨/下轨持续运行)
❌ 无止损保护
❌ 全仓进出风险大
❌ 两个版本逻辑相反，易混淆
