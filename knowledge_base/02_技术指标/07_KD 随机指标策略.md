## 1️⃣ 学习结论 (一句话)

**K 线上穿 D 线金叉买入，K 线下穿 D 线死叉卖出，超买超卖区反转信号。**

---

## 2️⃣ 核心内容

### 策略逻辑

```python
入场条件:
- K1 > D1 (当前金叉)
- K2 <= D2 (之前死叉或粘合)
→ 全仓买入

出场条件:
- K1 <= D1 (当前死叉或粘合)
- K2 > D2 (之前金叉)
→ 清仓出场
```

### KD 参数

```
N 周期：9 (默认)
M1 周期：3 (K 线平滑)
M2 周期：3 (D 线平滑)

标准参数：9,3,3
```

### KD 指标原理

```
K 线 (快速确认线):
- 反应敏感，波动大
- 测量当前收盘价在 N 周期内的相对位置

D 线 (慢速主干线):
- K 线的移动平均
- 反应较慢，更稳定

J 线 (超买超卖线):
- J = 3K - 2D
- 极端值判断超买超卖
```

### 超买超卖区

```
超买区：K/D > 80
- 价格可能回调
- 准备做空

超卖区：K/D < 20
- 价格可能反弹
- 准备做多
```

## 3️⃣ 总结

### 优点
✅ 经典震荡指标
✅ 超买超卖判断清晰
✅ 适合区间震荡市

### 缺点
❌ 趋势市频繁假信号
❌ 金叉死叉滞后
❌ 无止损保护
❌ 全仓进出风险大

### 与 ACCER 结合

```

- 测量价格变化加速度

- <0: 下跌加速

组合信号:
- KD 金叉 + ACCER>0 → 强买入信号
- KD 死叉 + ACCER<0 → 强卖出信号
```
