# DMI (ADX) 趋势强度指标策略

> 策略来源：聚宽 #79 | 学习时间：2026-03-18  
> 整理：旺财 | 费曼学习法笔记

---

## 1️⃣ 学习结论 (一句话)

**ADX 上升确认趋势强度，+DI 上穿-DI 做多，+DI 下穿-DI 做空。**

---

## 2️⃣ 核心内容

### 策略逻辑

```python
入场条件:
- ADX[-1] > ADX[-2] (ADX 上升，趋势增强)
- +DI[-1] > -DI[-1] (+DI 在-DI 上方，上涨方向)
- 当前无持仓或空仓
→ 全仓买入

出场条件:
- ADX[-1] < ADX[-2] (ADX 下降，趋势减弱)
- +DI[-1] < -DI[-1] (+DI 在-DI 下方，下跌方向)
- 当前有持仓
→ 清仓出场
```

### DMI 参数

```
ADX 周期：18 (原策略)
标准周期：14

组成:
- ADX: 平均趋向指数 (趋势强度)
- +DI: 上升方向指标
- -DI: 下降方向指标
```

### DMI 指标原理

```
ADX (Average Directional Index):
- 测量趋势强度，不指示方向
- ADX > 25: 强趋势
- ADX < 20: 震荡市
- ADX 上升：趋势增强
- ADX 下降：趋势减弱

+DI (Plus Directional Indicator):
- 测量上涨趋势强度
- +DI 上穿-DI: 买入信号

-DI (Minus Directional Indicator):
- 测量下跌趋势强度
- -DI 上穿+DI: 卖出信号
```

### 交易信号

```
强买入信号:
1. ADX > 25 (强趋势)
2. ADX 上升 (趋势增强)
3. +DI 上穿-DI (方向确认)

强卖出信号:
1. ADX > 25 (强趋势)
2. ADX 下降 (趋势减弱)
3. +DI 下穿-DI (方向反转)

震荡市 (不交易):
- ADX < 20
- 等待突破
```

## 3️⃣ 总结

### 优点
✅ 准确判断趋势强度
✅ 过滤震荡市假信号
✅ 适合趋势跟踪策略
✅ 可作为其他指标的过滤器

### 缺点
❌ ADX 滞后性强
❌ 震荡市频繁假信号
❌ 原策略无止损
❌ 全仓进出风险大
