## 📊 策略概述

**多因子选股策略** - 基本面 + 技术面双重筛选

**核心思路**: 
1. 基本面：低估值 (低 PB) + 低负债
2. 技术面：TRIX 趋势 + RSI 超买超卖

**适用市场**: 股票 (原策略), 可简化用于外汇

**时间框架**: 日线

---

## 🎯 策略逻辑

### 基本面筛选 (股票专用)

```python
筛选条件:
1. PB 比率 < 2 (低估值)
2. PB 比率 > 0 (排除负值)
3. 流动比率 > 1.2 (短期偿债能力)
4. 负债率低于中位数 (低负债)

目的：选择财务健康的低估股票
```

### 技术面信号

#### TRIX 指标

```
TRIX (Triple Exponential Smoothing Average)
- 三重指数平滑平均线
- 过滤短期波动，捕捉长期趋势

信号:
- TRIX > MATRIX (TRIX 的信号线) → 买入
- TRIX < MATRIX → 卖出

参数：N=12, M=20
```

#### RSI 指标

```
RSI (Relative Strength Index)
- 相对强弱指标
- 测量超买超卖

信号:
- RSI > 80 或 50>RSI>20 → 卖出 (超买或下跌趋势)
- RSI < 15 或 80>RSI>55 → 买入 (超卖或上涨趋势)

参数：N=6
```

### 交易规则

```
买入条件:
1. 基本面筛选通过
2. TRIX 买入信号 (-1)
3. RSI 买入信号 (-1)
4. 持仓数量 < MAX_OWN_NUM (默认 2 只)

卖出条件:
1. TRIX 卖出信号 (1)
2. 或 RSI 卖出信号 (1)

仓位管理:
- 最大持仓：2 只股票
- 单只股票上限：10,000 元
- 总资金使用上限：20,000 元
```

**结论**: 基本面因子不适用于外汇，但技术面因子 (TRIX+RSI) 可以借鉴

## 🔧 策略改进方案

### 方案 1: TRIX+RSI+AD X 三重确认

```python
入场条件:
1. TRIX > MATRIX (趋势向上)
2. RSI(9) 从<30 回升到>30 (超卖反弹)
3. ADX(14) > 25 (趋势强度)
4. +DI > -DI (方向确认)

出场条件:
1. TRIX < MATRIX
2. 或 RSI > 70
3. 或 止损触发

优点：多重过滤，信号质量高
缺点：入场机会较少
```

### 方案 2: 多时间框架版本

```
D1: 确定主趋势
- TRIX > MATRIX → 只做多
- TRIX < MATRIX → 只做空

H4: 生成交易信号
- RSI 超卖 + TRIX 金叉 → 入场

H1: 精确入场
- 等待回调入场
- 减少止损距离
```

### 方案 3: TRIX+RSI+AMA 组合

```
趋势确认：AMA 向上
动量确认：TRIX > MATRIX
入场时机：RSI 从超卖区回升

出场:
- TRIX 死叉
- 或 RSI 超买
- 或 吊灯止损

优点：三重确认，胜率高
缺点：复杂度高
```

## 📊 风险管理

### 仓位计算

```python
# 基于风险的仓位计算
风险金额 = 账户余额 × 2%
止损距离 = 入场价 - 止损价 (点数)
点值 = 每点价值

仓位 = 风险金额 / (止损距离 × 点值)

示例:
账户：$10,000
风险：2% = $200
EURUSD 入场：1.1000
止损：1.0960 (40 点)
点值：$10/手

仓位 = 200 / (40 × 10) = 0.5 手
```

### 止损设置

```
固定止损：2-3×ATR
技术止损：前低/前高
时间止损：入场后 20 根 K 线不盈利则出场

止盈:
- 第一目标：2R
- 第二目标：RSI > 70
- 追踪止损：吊灯止损 (3×ATR)
```
