# MACD 背离判断顶底策略

> 策略来源：聚宽 #08 | 学习时间：2026-03-18  
> 整理：旺财 | 费曼学习法笔记

---

## 1️⃣ 学习结论 (一句话)

**MACD 金叉买入、死叉卖出，用背离判断趋势反转。**

---

## 2️⃣ 核心内容

### 策略逻辑

```python
入场条件:
- MACD > 0 (快线在慢线上方)
- 当前无持仓或空仓
→ 全仓买入

出场条件:
- MACD < 0 (快线在慢线下方)
- 当前有持仓
→ 清仓出场
```

### MACD 参数

```
快线周期：9
慢线周期：24
信号线周期：9

标准 MACD 参数是 (12,26,9)
这个策略用了更敏感的 (9,24,9)
```

### 背离判断 (核心)

```
顶背离:
- 价格创新高
- MACD 未创新高
→ 趋势可能反转，准备做空

底背离:
- 价格创新低
- MACD 未创新低
→ 趋势可能反弹，准备做多
```

---

## 3️⃣ 总结

### 优点
✅ 简单易懂，经典指标
✅ 背离判断提前预警反转
✅ 适合趋势市

### 缺点
❌ 震荡市频繁假信号
❌ 参数 (9,24,9) 过于敏感
❌ 无止损，风险大
❌ 全仓进出，仓位管理激进
