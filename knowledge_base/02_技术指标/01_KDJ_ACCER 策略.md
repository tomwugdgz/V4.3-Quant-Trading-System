# KDJ + ACCER 过滤策略

> 策略来源：聚宽 #40 | 学习时间：2026-03-18

---

## 📊 策略概述

**核心思路**: KDJ 指标 + ACCER 加速度指标过滤，提高信号质量

**适用市场**: 股票、外汇、期货

**时间框架**: H4/D1 (外汇)

---

## 🧮 指标说明

### KDJ 指标

```
K 线：快速确认线
D 线：慢速主干线
J 线：超买超卖线

金叉：K 线上穿 D 线 → 买入信号
死叉：K 线下穿 D 线 → 卖出信号

超买区：K/D > 80
超卖区：K/D < 20
```

### ACCER 指标 (加速度)

```
原理：测量价格变化的加速度
> 0: 上涨加速
< 0: 下跌加速

作用：过滤震荡，确认趋势强度
```

---

## 🎯 交易规则

### 入场条件 (做多)

```
1. KDJ 金叉 (K 上穿 D)
2. KDJ 在超卖区 (< 30) 或从超卖区回升
3. ACCER > 0 (上涨加速)
4. 价格 > 关键均线 (如 MA20)

→ 四个条件同时满足才入场
```

### 入场条件 (做空)

```
1. KDJ 死叉 (K 下穿 D)
2. KDJ 在超买区 (> 70) 或从超买区回落
3. ACCER < 0 (下跌加速)
4. 价格 < 关键均线 (如 MA20)

→ 四个条件同时满足才入场
```

### 出场条件

```
止损：
- 固定止损：2-3×ATR
- 技术止损：前低/前高

止盈：
- 第一目标：2R (风险回报 1:2)
- 第二目标：KDJ 反向信号
- 追踪止损：吊灯止损
```

## ⚠️ 注意事项

### KDJ 的缺点

1. **震荡市假信号多** → 用 ACCER 过滤
2. **趋势市滞后** → 配合趋势指标 (ADX)
3. **极端行情失效** → 设置止损

### ACCER 的缺点

1. **噪音大** → 用均线平滑
2. **滞后性** → 缩短周期
3. **假突破** → 配合成交量/波动率

## 🔧 策略改进方向

### 改进 1: 三重确认

```
KDJ 信号 + ACCER 确认 + ADX 趋势强度
→ 三个条件同时满足
```

### 改进 2: 多时间框架

```
D1: 确定主趋势方向
H4: 生成交易信号
H1: 精确入场点位
```

### 改进 3: 动态参数

```
高波动期：延长 KDJ 周期 (14→21)
低波动期：缩短 KDJ 周期 (9→7)
```
